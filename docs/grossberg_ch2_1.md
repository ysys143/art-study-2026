## 1. 역사적 배경: 세 개의 N의 탄생

### 19세기: 물리학자들이 심리학자이던 시대

19세기에는 위대한 물리학자들이 동시에 위대한 심리학자이기도 했습니다. 이 놀라운 전통은 현대 과학에서 거의 찾아볼 수 없습니다.

**다학문적 거장들의 예시**:

| 과학자 | 물리학 기여 | 심리학/뇌과학 기여 |
|--------|-------------|-------------------|
| **Hermann von Helmholtz** | 에너지 보존 법칙 발견 | 시각/청각 지각의 기초 확립 |
| **Clerk Maxwell** | 전자기 이론 (Maxwell 방정식) | 삼색 이론 제안 (색채 수용체 예측) |
| **Ernst Mach** | 마하 수 (초음속 비행) | 마흐 대역 (시각 대비) 발견 |

### 인지 불투명성의 발견

그런데 20세기 초반, 이 다학문적 전통이 갑자기 끊어집니다. 왜 그랬을까요?

1933년, 알버트 아인슈타인은 벨기에 엘리자베스 여왕에게 보낸 편지에서 이렇게 썼습니다:

> "우리 대부분은 자기 내부보다 외부를 더 좋아합니다. 후자의 경우, 우리는 **아무것도 의미하지 않는 어두운 구멍**만 봅니다."

이 **"어두운 구멍"**이 바로 **인지 불투명성(cognitive impenetrability)**입니다. 우리의 일상 경험을 조절하는 신경 계산은 의식에 접근 불가능합니다. 이 불투명성 덕분에 우리는 복잡한 신경 기계를 인식하지 않고도 지각, 감정, 사고의 세계에서 자연스럽게 행동할 수 있습니다.

---

### 세 개의 N: 비선형(Nonlinear), 비국소적(Nonlocal), 비정상적(Nonstationary)

헬름홀츠의 시각 지각 연구는 뇌 과학이 **새로운 수학**을 필요로 한다는 것을 보여주었습니다.

#### 색채 인식에서의 비선형성 발견

**뉴턴의 발견 (1704)**: 흰색 빛은 모든 파장의 동일한 강도로 구성

**헬름홀츠의 혁신적 통찰**:
- 장면의 평균 색상은 **감쇠되어(desaturated)** 인지됨
- 이를 설명하려면 **장거리 상호작용(비국소적)** 필요
- "광원을 할인"하는 과정은 **나눗셈**을 포함 → **비선형적**

> **핵심**: 덧셈과 뺄셈이 아닌 연산(나눗셈 등)을 포함하는 모든 과정은 비선형적입니다.

#### 무의식적 추론과 비정상성

헬름홀츠는 **무의식적 추론(unconscious inference)** 교리를 발전시켰습니다: 우리는 과거 학습을 기반으로 기대하는 것을 지각합니다.

이는 현대 용어로 **적응 공명 이론(Adaptive Resonance Theory)**의 예고였습니다:
1. 하향식(bottom-up) 학습: 입력이 LTM 가중치를 조정
2. 인식 카테고리 형성
3. 상향식 기대의 하향식 매칭
4. 협력적-경쟁적 매칭

이 과정들은 시간에 따라 LTM 흔적이 변경되므로 **비정상적(nonstationary)**입니다.

#### 19세기 물리학의 한계

**19세기 물리학**: 선형적(linear), 국소적(local), 정상적(stationary) 수학

**뇌 과학의 요구**: 비선형적, 비국소적, 비정상적 수학

이 **근본적 불일치**가 심리학 및 신경생물학 데이터를 설명할 수 있는 이론의 부재를 가져왔습니다. 이것이 바로 19세기 말의 **지적 위기(intellectual crisis)**였습니다.

## 2. 뇌가 마음을 만드는 방식

### 다중 수준의 설명: 숲, 나무, 잎을 동시에 보기

뇌 이론의 난제: **행동, 신경망, 개별 뉴런** — 세 가지 수준을 동시에 이해해야 합니다.

> 비유: "전체 숲, 개별 나무들, 그리고 모든 나무의 잎들에 대해 알려진 모든 것을 한 번에 마음에 담기"

이 업적을 수행하려면 **과학적 언어**가 필요합니다. 수학만이 그러한 복잡성을 직접적이고 모호하지 않은 용어로 표현할 정밀도를 가지고 있습니다.

### 미분 방정식의 선택

**미분 방정식**이 시간에 따라 사건이 진화하는 방식을 설명하는 보편적 형식주의입니다:

$$\frac{dx}{dt} = f(x, t)$$

이 형식을 사용하면:
- 개별 뉴런의 역학
- 신경막 상호작용
- 창발적 행동 특성

세 수준을 동시에 표현할 수 있습니다.

### 1957년: 여정의 시작

1957년, 다트머스 칼리지 1학년생 스티븐 그로스버그는 **Introductory Psychology** 수업에서 기억 법칙을 발견했습니다.

**발견 내용**:
- 단기 기억(STM), 중기 기억(MTM), 장기 기억(LTM)의 법칙
- 이 법칙들의 변형은 오늘날 대부분의 계산 신경과학자들이 여전히 사용

이 상호작용하는 법칙들이 인간 학습 데이터를 어떻게 설명할 수 있는지 보여주며, **신경망의 창발적 특성이 관찰 가능한 행동과 일치**하는 시스템을 소개했습니다.

### 자율적 적응의 중요성

핵심 통찰: 우리의 마음은 환경 조건이 변경되었음을 알려주지 않아도 **스스로 변화하는 세계에 적응**합니다.

이것이 의미하는 것:
- 예상치 못한 사건으로 가득 찬 변화하는 세계
- 실시간 자율적 적응
- 정적 데이터 곡선을 연속적으로 시간에 따라 진화하는 기저 정신 과정으로 변환

### 행동 데이터에서 신경망으로

1957년, 심리학적 데이터 분석에서 자연스럽게 **신경망 모델**이 도출되었습니다:

```심리학적 학습 데이터 분석
    ↓
동적 시스템 방정식 발견
    ↓
신경과학적 사실과 일치 확인
    ↓
신경도ctrine의 "발견"
```

놀랍게도, 신경과학을 전혀 모르는 상태에서 심리학적 데이터를 사용하여 **신경 독트린의 기본 사실을 파생**시켰습니다!

## 3. 연속 학습과 시간의 역설

### 시리즈 포지션 곡선 (Bow Curve)

연속 학습에서 가장 기본적인 현상은 **bow-shaped 곡선**입니다:

```
정확도
↑
│      ● (초기 효과)
│     / \
│    /   \      ● (중간 항목)
│   /     \    / \
│  /       ●───/   \      ● (최신 효과)
│ /       /        \
│●───────/          \──────●
└─────────────────────────→
   첫 번째   중간    마지막
```

| 위치 | 현상 | 원인 |
|------|------|------|
| **시작** | 가장 높은 기억 | 첫 학습 · 주의 집중 |
| **중간** | 가장 낮은 기억 | 대비 부족 · 맥락 약화 |
| **끝** | 높은 기억 | 최근 학습 · 단기 기억 |

Grossberg는 이 bow 곡선을 다음 동적 시스템으로 모델링했습니다:

$$\frac{dx_i}{dt} = -A_i x_i + f(\sum_{j<i} x_j) I_i - g(\sum_{j>i} x_j) I_i$$

여기서 각 항목은 앞에서 오는 흥분과 뒤에서 오는 억제 사이의 균형에서 결정됩니다.

---

### Hovland의 실험: 시간적 간격의 효과

1940년대, **Carl Hovland**는 연속 학습에서 시간적 간격의 영향을 연구했습니다.

#### 두 가지 간격 유형

| 간격 타입 | 정의 | 효과 |
|-----------|------|------|
| **실험 내 간격** | 세션 내 항목 간 간격 | 간격 증가 → 기억 정확도 증가 |
| **실험 간 간격** | 세션 사이 간격 | 간격 증가 → 장기 기억 향상 |

Grossberg의 해석:
- **실험 내 간격**: 앞 항목의 잔류 활동 → 뒤 항목 억제 → 적절한 간격으로 억제 감소
- **실험 간 간격**: 세션 간 휴식 → 잔류 활동 소멸 → 새로운 패턴 저장 가능

---

### "시간 역방향" 영향의 역설

연속 학습에서 **미래의 정보가 과거의 학습에 영향을 미치는** 것처럼 보이는 현상:

- 후반 항목 학습이 전반 항목 기억에 영향
- 예상된 패턴이 실제 학습 효율 변경

이 역설은 **순환적 인과관계(recursive causality)**로 설명됩니다:

```
과거 활동 → 현재 활동에 영향
    ↓
현재 활동 → 미래 활동 예측
    ↓
예측된 미래 → 현재 활동 수정
    ↓
수정된 현재 → 과거 활동 재해석
```

수학적 표현:

$$x_i(t) = f\left(\sum_{j<i} w_{ji} x_j(t), \sum_{k>i} v_{ki} \frac{dx_k}{dt}(t+\tau)\right)$$

---

### 리스트 청킹(List Chunking)과 작업 기억

인간은 긴 리스트를 단순히 순서대로 기억하지 않습니다. **의미적 그룹화(청킹)**를 통해 정보를 청킹합니다:

```
원본: A-B-C-D-E-F-G-H-I-J (10개 항목)
청킹: [A-B-C] [D-E-F] [G-H-I] [J] (4개 청크)
```

**청킹의 장점**:
- 인지 부하 감소 (7±2 용량 초과 방지)
- 의미적 처리 및 구조화
- 학습 효율성 증가

Grossberg의 작업 기억 모델:

```
입력층 → 감각 기억 버퍼 → 단기 기억 핵심 → 장기 기억 연결
```

수식:

$$\frac{dS_i}{dt} = -\alpha S_i + \sum_j W_{ij} I_j + \sum_k V_{ik} L_k$$

## 4. STM과 LTM의 발견과 상호작용

### HM 환자 사례 (Scoville와 Milner, 1957)

**획기적 발견**: STM과 LTM의 특성은 분리될 수 있다.

HM 환자는 해마(hippocampus) 양측 제거 수술 후:
- STM 유지: 최근 사건들을 STM에 저장 가능
- LTM 결손: 주의 산만해지면 모든 흔적 소멸
- **프로시저럴 기억은 보존**: 의식적 기억 없이도 기술 습득 가능

이 발견은 뇌의 **계산적 보완 피질 처리 스트림**을 보여줍니다.

---

### STM과 LTM의 상호작용

1960년대 Grossberg의 분석: 학습의 기능적 단위는 개별 세포 쌍 간의 흔적이 아닌, 네트워크 전반에 걸쳐 **분산된 패턴(patterns) 또는 벡터(vectors)**의 LTM 흔적입니다.

**LTM 법칙의 이중 성격**:
- 증가: LTM 흔적이 큰 STM 활동과 일치할 때 (**헵시안, Hebbian**)
- 감소: LTM 흔적이 작은 STM 활동과 일치할 때 (**반-헵시안, anti-Hebbian**)

---

### 헵스 가설의 한계

헵스(1949)는 "세포 A가 세포 B를 흥분시킬 때 A의 효능이 증가한다"고 가정했습니다.

**문제점**:
1. **잘못된 계산 단위 선택**: 단일 경로 vs 분산된 LTM 패턴
2. **유한한 상한 문제**: LTM 흔적이 상한값에 도달하면 추가 학습 불가

---

### 게이트드 가장 급강하 학습

**개념**: 학습은 전신경 STM "샘플링" 신호가 충분히 활성화될 때만 발생

수학적 표현:

$$\frac{dz_{ki}}{dt} = \begin{cases}
0 & \text{if } f_k^{sample}(x) \leq 0 \\
-f_k^{sample}(x) [h_i(x) - z_{ki}] & \text{if } f_k^{sample}(x) > 0
\end{cases}$$

**작동 원리**:
- **게이트(장벽)**: $f_k^{sample}(x) > 0$일 때만 학습 가능
- **가장 급강하(steepest descent)**: $h_i(x) - z_{ki}$를 따라 LTM 흔적이 수렴
- **비대칭**: LTM 흔적이 증가하거나 감소 가능

이 모델은 20년 후 신경생리학적으로 확인되었습니다.

---

### 샌팅 모델과 막 방정식

Grossberg는 막 방정식을 모르는 상태에서 순수한 사고 실험을 통해 **샌팅 모델**을 도출했습니다.

**막 방정식**:

$$C\frac{\partial V}{\partial t} = (V^+ - V)g^+ + (V^- - V)g^- + (V^p - V)g^p$$

각 항은 "전압 차이 × 전도도" 형태로, **곱셈적(shunting) 구조**를 가집니다.

**Additive Model vs Shunting Model**:

| 특징 | Additive Model | Shunting Model |
|------|----------------|----------------|
| 수식 | $\frac{dx}{dt} = -Ax + \sum f z + I$ | $\frac{dx}{dt} = -Ax + (B-x)[\cdots] - (Ex+F)[\cdots]$ |
| 상한/하한 | 없음 | 자동으로 $[-F/E, B/C]$ 범위 |
| 제어 방식 | 덧셈적 | 곱셈적 (자동 이득 제어) |

---

### MTM (Medium-Term Memory)

**시간적 차이**:
- **STM**: 가장 빠른 변화
- **MTM**: STM보다 더 느리게 변화
- **LTM**: MTM보다 더 느리게 변화

**MTM 수식**:

$$\frac{dy_{kj}}{dt} = -D_{kj} y_{kj} + D_{kj} f_j(x_j) z_{kj}$$

**전달자 게이트(transmitter gate)**: $f_j(x_j) z_{kj}$에 의해 활성화된 신호를 곱하며, **습관적(habituative)** 특성을 가집니다.

## 5. 이진 모델에서 연속 모델로

### McCulloch-Pitts 모델 (1943): 이진 스파이크의 시작

신경망 연구의 세 근원 중 하나인 **이진 모델(Binary models)**:

**특징**:
- 이산 시간(discrete time)에서 작동
- 임계값 논리(threshold logic) 시스템
- 뉴런 활동 $x_i$는 이진 신호(+1, 0, -1) 방출
- `sgn` 함수 사용

**문제점**: -1 신호는 생물학적으로 설명 어려움

---

### Eduardo Caianiello의 기여 (1961)

**STM 방정식** 제안:
- 과거의 다양한 시점에서의 활동에 영향
- **생물학적으로 타당한 이진 신호**: 활동이 양수일 때 +1, 그 외에는 0
- McCulloch-Pitts 모델의 음수 신호 문제 해결

---

### Frank Rosenblatt의 Perceptron (1958)

- 연속 시간(continuous time)에서 진화
- 자발적으로 감쇠할 수 있는 활동
- 비제로 임계값 위에서 이진 신호 생성

**학습 문제**:
- STM과 LTM 상호작용이 분리되어 분석 단순화
- 자율적인 실시간 작동 불가

---

### 역전파(Backpropagation)의 역사와 한계

**독립적 발견**:
- 1972년: Shun-Ichi Amari
- 1974년: Paul Werbos (하버드 박사 학위 논문)
- 1986년: Rumelhart, Hinton, Williams

**기술적 기원**: 최급하강법(steepest or gradient descent) - Carl Friedrich Gauss(1814) 또는 그 이전

**생물학적 문제점**:
- **비국소적 가중치 이동(non-local transport of weights)**
- 뇌에서와 달리 가중치가 알고리즘의 한 지점에서 계산되어 다른 지점으로 이동

**딥러닝의 한계**:
- 재앙적 잊어버림(catastrophic forgetting)
- 일반화 성능 부족(poor generalization)
- 설명 가능성 문제(Explainable AI)

2017년 Hinton의 자기 비판:
> "역전파에 대해 심각히 의심스럽다. 뇌가 그렇게 작동하는 것 같지 않다."

---

### Anderson의 Brain-State-in-a-Box (BSB)

James Anderson의 패턴 인식 모델:
- 선형 방정식에서 활동 결정
- 연속적이고 이진적 특성 결합
- 출력이 중간 범위에서는 선형, 매우 작거나 크면 포화

이름의 유래: 뇌 상태가 상자 안에 담긴 듯, 상자 벽(포화)에 도달할 때까지 활동 변화

---

### Kohonen의 Self-Organizing Maps (SOM)

Teuvo Kohonen의 전이: 선형대수에서 생물학적 모델로

**작동 원리**:
1. 적응 필터(adaptive filter) 출력의 경쟁
2. 가장 큰 총 입력을 받는 셀이 승자로 선택
3. 승자 카테고리가 들어오는 데이터를 분류

**한계**:
- 매우 단기 기억(STM) 부재
- 중기 기억(MTM) 결여
- ART 내에서 작동 시 안정성-가소성 딜레마 해결

---

### 생물학적 근거: Hartline-Ratliff와 Hodgkin-Huxley

**Hartline-Ratliff 방정식**: 말굽게비(limulus) 망막 모델
- 재귀 억제(recurrent inhibition)가 망막의 순환성 감소
- 연속 임계값 논리 시스템
- Additive Model의 선구자

**Hodgkin-Huxley 방정식**: 오징어 거대 축삭 모델
- 신경 충파(propagation)의 수학적 모델링
- 개별 뉴런에 초점 (Grossberg의 신경망 초점과 대조적)

---

## 6. 왜 신경망인가

### Hartline-Ratliff-Miller 모델: 투구게 망막에서 시작되는 이야기

1967년, Hartline이 노벨상을 수상한 해와 정확히 동일한 시기에 저자는 록펠러 대학교에서 수학 박사 학위를 받았습니다.

**놀라운 일치**: Hartline-Ratliff-Miller 모델은 저자가 발견한 신경망 네트워크의 **특수 사례**였습니다!

투구게 망막의 감산적 측면 억제(subtractive lateral inhibition) 패턴:

$$r_i = e_i - \sum_{j \neq i} k_{ij} r_j$$

---

### 전역 한계 정리: 수학적으로 보안된 안정성

저자가 증명한 정리들은 임의로 큰 수의 뉴런을 가진 네트워크에 적용 가능:

$$\lim_{t \to \infty} \| \mathbf{x}(t) \| < \infty$$

신경망이 시간이 지나도 폭주하지 않고 안정된 상태를 유지함을 수학적으로 보장합니다.

---

### 기능적 단위: 단일 스파이크가 아닌 분산된 패턴

**인식의 패러다임 전환**: 개별 뉴런의 활동이 아니라 **네트워크 전체에 걸쳐 분산된 활동 패턴**이 기능적 단위입니다.

> **중요한 발견**: "기능적 성공을 지배하는 단위는 개별 뉴런의 단일 스파이크, 스파이크 시열, 또는 스파이크 폭발이 아닌, **네트워크 및 시스템 수준의 분산된 패턴**입니다."

**분산된 활동 패턴의 장점**:
1. 정보 병렬 처리
2. 내성(Robustness): 일부 뉴런 손상 시 기능 유지
3. 표현력: 기하급수적으로 증가하는 정보 인코딩 용량
4. 유연성: 환경 변화에 빠른 적응

---

### 다중 스케일 피드백 네트워크

**다중 스케일(multiple-scale)**의 의미:

| 스케일 | 범위 | 시간 단위 |
|--------|------|-----------|
| **미시적** | 개별 뉴런, 시냅스 | 수 밀리초 |
| **중간** | 뉴런 회로, 기능적 그룹 | 밀리초~초 |
| **거시적** | 전체 시스템, 네트워크 | 초~분 |

**피드백의 역할**:
- 자기 조절: 시스템 내부 상태 감시
- 예측 오류 보정: 기대치와 실제 입력 불일치 조정
- 적응적 학습: 경험에 따른 구조적 변화

이러한 구조는 **보편적 발달 코드(universal developmental code)**의 일부입니다.

---

## 7. 노이즈-포화 딜레마와 그 해결

### 문제: 무한 입력 범위의 딜레마

뇌의 뉴런은 다음과 같은 기본적 딜레마에 직면:

1. **작은 입력**: 노이즈에 묻혀서 감지되지 않음
2. **큰 입력**: 시스템이 포화(saturation)되어 패턴 정보 소실

이것이 바로 **노이즈-포화 딜레마(Noise-Saturation Dilemma)**입니다.

---

### 전통적 해결책의 한계

#### 1. 시그모이드 함수

$$\text{output} = \frac{1}{1 + e^{-x}}$$

- **문제**: 모든 뉴런이 포화될 때 패턴 정보 완전 소실

#### 2. 감마 보정

$$\text{output} = \text{input}^\gamma$$

- **문제**: 입력 간 비율이 왜곡됨 (상대적 관계 보존 불가)

---

### 해결책: 곱셈적 제어와 자동 이득 제어

Grossberg의 핵심 통찰: 입력이 활동을 **덧셈적으로**가 아니라 **곱셈적으로** 제어해야 한다!

**Shunting 모델**:

$$\frac{d}{dt}x_i = -A_i x_i + (B_i - C_i x_i)[\cdots] - (E_i x_i + F_i)[\cdots]$$

- 흥분 입력이 $(B_i - C_i x_i)$와 **곱해짐**
- 억제 입력이 $(E_i x_i + F_i)$와 **곱해짐**

**자동 이득 제어(AGC)의 원리**:
- 활동이 작을 때: $(B - x_i)$가 큼 → **높은 게인**
- 활동이 클 때: $(B - x_i)$가 작음 → **낮은 게인**

---

### 상호작용 없는 shunting의 문제

$$\frac{d}{dt}x_i = -A x_i + (B - x_i) I_i$$

평형 해석: $x_i = \frac{B I_i}{A + I_i}$

**문제**: 입력 $I$가 커지면 $x_i \approx B$ → **모든 뉴런이 포화**, 패턴 정보 완전 소실

---

### 진정한 해결: On-center Off-surround 해부학

패턴 정보를 보존하려면 **셀 간의 상호작용**이 필요합니다:

$$\frac{d}{dt}x_i = -A x_i + (B - x_i) I_i - x_i \sum_{k \neq i} I_k$$

**평형 해석**:

$$x_i = \frac{B I_i}{A + \sum_k I_k} = \frac{B I_i}{A + I}$$

**결과의 의미**:
1. **비율 보존**: $x_i = \frac{B I}{A + I} \cdot \theta_i$ (여기서 $\theta_i = I_i/I$)
2. **포화 방지**: $I$가 아무리 커도 $x_i \to B \theta_i$ (패턴 유지)
3. **자동 정규화**: 총 활동 $\sum x_i \leq B$

---

### Brightness Constancy와 Reflectance

On-center off-surround 구조는 **반사율(reflectance)**을 계산:

$$\theta_i = \frac{I_i}{\sum_j I_j}$$

이것이 **밝기 일관성(Brightness Constancy)** 원리입니다:
- 물체의 실제 반사율은 조명 조건에 무관
- 우리는 $\theta_i$ 값을 계산하여 일관된 지각

**Weber 법칙의 자연스러운 결과**:
- 배경 조명이 바뀌어도 상대적 민감도 유지
- 이동 특성(shift property) 발생

---

### Simultaneous Contrast (동시 대비)

같은 회색 점이:
- 밝은 배경에서는 어둡게 보임
- 어두운 배경에서는 밝게 보임

수학적 설명:

$$x_i = \frac{B I_i}{A + I}$$

- 밝은 배경 ($I$ 큼): 분모 큼 → $x_i$ 작음
- 어두운 배경 ($I$ 작음): 분자 작음 → $x_i$ 큼

---

## 요약: 2장 전반부의 핵심 메시지

```
역사적 분열 (물리학 ↔ 심리학)
    ↓ 세 개의 N의 발견
비선형, 비국소적, 비정상적 수학의 필요성
    ↓
뇌가 마음을 만드는 방식 발견
    ↓ 행동 데이터에서 출발
STM, LTM, MTM의 상호작용
    ↓ 게이트드 가장 급강하 학습
이진 모델 → 연속 비선형 모델
    ↓
신경망의必要性: 분산 패턴 처리
    ↓ 다중 스케일 피드백
노이즈-포화 딜레마의 해결
    ↓ On-center Off-surround
자동 이득 제어, 비율 보존, Weber 법칙
```

이 서사는 역사적 지적 위기에서 출발하여, 뇌의 근본적인 계산 원리를 단계적으로 밝혀가는 과정입니다. 다음 후반부에서는 이 원리들이 막 방정식, Weber 법칙, 적응 수준, 노이즈 억제, 패턴 매칭 등으로 어떻게 발전하는지 다룹니다.
