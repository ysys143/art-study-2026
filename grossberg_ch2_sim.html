Additive vs Shunting Model
Additive Model(Fig 2.9)은 입력이 활동에 그냥 더해지므로, 활동에 상한/하한이 없다. Shunting Model(Fig 2.10)은 입력이 (B - x)와 곱해지므로, 활동이 자동으로 [0, B] 범위에 묶인다. 같은 입력을 넣었을 때 두 모델의 반응이 어떻게 다른지 비교한다.
Additive
Shunting
뉴런 i의 막 활동(membrane activity). 막 전압 V를 [0, B] 범위로 정규화한 상태 변수. 피질 세포의 발화율(firing rate)에 대응.
뉴런 i로 들어오는 흥분성 시냅스 전류. 광수용체·시냅스 전도도 등 외부 자극을 대표. Shunting에서 (B−x)와 곱해져 자동 이득 제어(AGC)를 수행.
수동 막 전도도에 비례하는 감쇠율. 자극이 없을 때 세포를 정지 전위(resting potential)로 복귀시키는 생리적 누설 전류(leak current)를 모델링.
흥분성 포화전압(reversal potential). 세포 내부보다 높은 Na⁺/Ca²⁺ 평형전위에 해당. x→B일수록 (B−x)→0, 즉 입력이 더 들어와도 활동이 포화되지 않고 비율을 유지.

Medium-term & Long-term Memory
MTM(Habituative Transmitter Gate): 신경전달물질이 활동에 의해 소진되고, 비활성 시 회복된다. LTM(Gated Steepest Descent Learning): 시냅스 가중치가 사전/사후 세포의 상관에 따라 학습된다.
MTM: Transmitter Gate
LTM: Learning Rule
Parameters
0.0
5.0
MTM: Transmitter y(t) — depletes when active
LTM: Weight z(t) — learns toward h(x)
MTM: f(x)가 0이면 y는 K(최대)로 회복. f(x)가 크면 y가 소진(habituation). LTM: f(x)가 0이면 학습 정지. f(x) > 0이면 z가 h(x)를 향해 수렴(gated learning).
Problem
유한한 뉴런(활동 상한 B)으로 무한한 입력 범위를 어떻게 처리하는가? 셀 간 상호작용이 없으면 입력이 커질수록 모든 셀이 B로 포화되어 패턴(비율) 정보가 사라진다. On-center off-surround 네트워크는 입력이 아무리 커져도 비율을 보존한다.
