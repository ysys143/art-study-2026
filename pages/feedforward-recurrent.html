1. 비순환 네트워크: 기억 없는 즉시 평형
비순환 네트워크의 출력은 입력의 대수적 함수입니다. 미분방정식을 풀 필요 없이 입력 Ii가 주어지면 즉시 평형값 xi*이 결정됩니다. 총 입력 I = ΣIk가 분모에 있으므로 정규화(normalization)가 자동으로 이루어지고, 활동의 총합은 항상 B 이하로 제한됩니다.
→ 현재 FF 평형: x₁* = 0.00
그러나 입력이 사라지면(I=0) 출력도 즉시 0이 됩니다. 기억 능력이 없습니다.
2. 순환 네트워크: 시간에 따른 수렴
순환 네트워크는 미분방정식으로 기술됩니다. 각 항의 역할:
수동적 감쇠 (passive decay)
외부 자극 없으면 활동이 지수적으로 소멸
분로 흥분 (shunting excitation)
활동이 B에 가까울수록 흥분 효과 감소 → 포화 방지
측면 억제 (lateral inhibition)
경쟁 셀의 활동이 클수록 자신의 활동 억제
3. 입력 제거 후: Attractor와 단기 기억(STM)
입력이 사라져도(Ii=0) 순환 연결 덕분에 활동이 즉시 사라지지 않습니다. 경쟁 항 c · xi · xj가 핵심입니다: 더 강한 셀이 약한 셀을 억제하면서 자기 자신은 억제를 덜 받아 비대칭 상태가 안정적으로 유지됩니다.
핵심 메커니즘: 셀 1이 셀 2보다 크면 (x₁ > x₂) → 셀 2가 받는 억제(c · x₂ · x₁)가 더 강해 → 셀 2 감소 → 셀 1이 받는 억제 약화 → 셀 1 유지. 이 양성 피드백 루프가 attractor를 형성합니다.
4. 끌개(Attractor)의 유형과 Winner-Take-All
Winner-take-all은 끌개의 특수한 유형입니다. 순환 네트워크의 ODE가 수렴하는 안정 고정점(stable fixed point)이 곧 끌개인데, 경쟁 계수 c의 크기에 따라 끌개의 형태가 달라집니다:
c = 0: 독립 끌개
각 셀이 독립적으로 평형 도달
c 작음: 공존 끌개
부분 대비(partial contrast) — 두 셀 모두 활성
c 큼: 선택 끌개
Winner-take-all — 범주적 선택
왜 WTA도 끌개인가? 상태 공간에서 (B, 0)과 (0, B)는 각각 안정 고정점입니다. 초기 입력의 미세한 차이가 시스템을 어느 끌개로 끌어당길지 결정합니다. Grossberg는 이를 "choice attractor"라고 부릅니다 — 시스템이 노이즈 속에서 범주적 선택을 내리는 메커니즘이자, noise-saturation dilemma의 해결책입니다.
→ 슬라이더로 c를 높여보고, 펄스 입력 후 끌개 형태의 변화를 관찰해 보세요. c가 클수록 두 셀의 활동 차이가 극대화되는 것을 확인할 수 있습니다.
