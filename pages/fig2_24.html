Figure 2.24: 신경생리학의 막 방정식

Membrane Equation
막 방정식과 Shunting 네트워크
Hodgkin-Huxley 스타일의 신경생리학 막 방정식(membrane equation)이 Grossberg의 shunting on-center off-surround 네트워크와 수학적으로 동일한 형태임을 보여줍니다. 각 이온 채널의 전도도(conductance)가 역전 전위(reversal potential)에서의 구동력을 조절하며, 이것이 자동 이득 제어(automatic gain control)와 silent inhibition의 물리적 근거입니다.
막 방정식 (Membrane Equation)
C = 막 용량(capacitance)  |  V = 막 전위  |  V⁺/V⁻/Vᵖ = 역전 전위  |  g⁺/g⁻/gᵖ = 전도도
신경망으로의 변수 대응
생리학 변수 신경망 변수 값 / 의미
C (capacitance) 1 단위 용량으로 정규화
V (membrane potential) xᵢ (활동) 셀 i의 활동 수준
V⁺ (excitatory reversal) B (saturation) 흥분 포화 상한. 활동이 B에 가까울수록 흥분 입력 효과 자동 감소
V⁻ = Vᵖ (inhibitory & passive reversal) 0 Silent inhibition — 억제 역전 전위 = 0. 억제는 x를 0으로만 끌어당기며 0 이하로 내려가지 않음
g⁺ (excitatory conductance) Iᵢ (on-center input) 셀 i에 대한 흥분 입력
g⁻ (inhibitory conductance) J = Σk≠i Iₖ (off-surround) 주변 셀들의 억제 입력 합계
gᵖ (passive conductance) A (decay) 수동 누수 전도도 — 입력 없으면 x → 0으로 감쇠
대입 결과: On-Center Off-Surround Shunting 방정식
막 방정식에 V⁺=B, C=1, V⁻=Vᵖ=0, g⁺=Iᵢ, g⁻=J, gᵖ=A 를 대입하면 그대로 도출됩니다.
회로 다이어그램 & Silent Inhibition
억제 유형 비교
Silent Inhibition (V⁻ = 0)
억제항 = −x·J
x = 0이면 억제 전류 = 0
→ 활동은 항상 x ≥ 0
→ 억제 = 이득 분할(gain division)
→ 비선형 대비 향상
Hyperpolarizing (V⁻ = −C)
억제항 = −(x+C)·J
x = 0이어도 억제 전류 ≠ 0
→ 활동이 x < 0 가능
→ 억제 = 뺄셈(subtraction)
→ 단순 역치 이동
핵심: Silent inhibition (V⁻ = 0)이면 억제 전류 = −x·J, 즉 현재 활동 x가 작을수록 억제 효과도 함께 줄어듭니다. 이것이 분로(shunting)의 물리적 의미입니다 — 억제가 활동을 나누어 이득을 제어하지, 고정량을 빼지 않습니다. 그 결과 활동은 항상 비음수(non-negative)이며 입력의 비율이 보존됩니다.

핵심 통찰
막 방정식 = Shunting 방정식: Hodgkin-Huxley 막 방정식에 V⁺=B, V⁻=Vᵖ=0, C=1 을 대입하면 Grossberg의 on-center off-surround 네트워크 방정식이 정확히 도출됩니다. 신경생리학과 신경망은 동일한 수학을 공유합니다.
자동 이득 제어
흥분 전류 = (B−x)·Iᵢ: x가 B에 가까울수록 흥분 효과가 자동으로 0이 됩니다. 전압이 역전 전위에 접근할수록 전류가 감소하는 물리 법칙 그대로입니다.
Silent Inhibition (분로)
억제 전류 = −x·J: x = 0이면 억제가 가해져도 아무 변화가 없습니다. 이것이 활동이 항상 0 이상을 유지하는 이유이고, 억제가 이득을 분할(divide)하는 방식으로 동작하는 이유입니다.
비율 보존 (Ratio Invariance)
x* = BI/(A+I+J). I가 어떤 값이든 x* < B. 여러 셀의 x*는 입력 비율을 보존합니다 — Weber의 법칙과 연결됩니다.
