Medium-term & Long-term Memory
앞서 Shunting STM 모델은 노이즈-포화 딜레마를 해결했다. 그러나 STM만으로는 두 가지가 빠져 있다:

① 신호가 계속 오면 어떻게 되는가? — 생물학적 시냅스에서 신경전달물질(vesicle)은 무한하지 않다. 반복 자극에 소진되고(habituation), 쉬면 회복된다. 이 과도적(transient) 반응을 모델링하기 위해 전달물질 게이트 y의 동역학이 필요하다. 생산-소모의 질량 작용(mass action) 원리에서 직접 도출된다.

② 경험에 의해 연결 강도가 어떻게 바뀌는가? — 시냅스 가중치 z는 고정이 아니라 학습에 의해 변해야 한다. Grossberg는 오차 함수 E=½(h−z)²의 steepest descent(최급하강법)에 사전 세포 활동 f(x)를 게이트로 곱하여 LTM 학습 규칙을 유도했다. 활동이 없으면 학습이 정지되므로 기존 기억이 보존된다(gated learning).
Shunting 모델에서의 위치
Additive: D · z
가산 모델에서 시냅스 출력은 D·z·f(x). LTM 가중치 z가 구조 상수 D와 곱해져 흥분성 합산에 직접 더해진다. y(MTM) 게이팅 없음.
Shunting: D · y · z
Shunting 모델에서 시냅스 출력은 D·y·z·f(x). y(transmitter, MTM)가 z(weight, LTM)를 곱셈적으로 게이트하여 4중 곱으로 자동 이득 제어.
방정식 분해
뉴런 j → 뉴런 i 하나의 시냅스 연결에 대한 방정식이다. 신호를 보내는 쪽이 j(사전 세포, presynaptic), 받는 쪽이 i(사후 세포, postsynaptic).
f(x) = 사전 세포 j의 신호 함수(signal function). 막 전위 xj를 입력받아 발화율(firing rate)을 출력한다. 보통 sigmoid 형태로, x가 임계값 이하이면 거의 0, 이상이면 포화 수준까지 증가한다. MTM과 LTM 모두에서 게이트 역할을 한다: f(x)=0이면 전달물질 소모도 없고(y 회복), 학습도 정지(z 고정). f(x)>0이면 전달물질이 소진되고(habituation), 동시에 가중치가 목표를 향해 이동(learning).
h(x) = 시냅스 j→i의 목표 가중치. sigmoid 같은 활성화 함수가 아니다. 자세한 설명은 아래 별도 카드 참조.
실제 네트워크에서 뉴런 i에 연결된 모든 j마다 독립된 yji, zji가 존재한다. 아래 시뮬레이션은 단일 연결 하나를 추적하므로 첨자를 생략하고 y, z로 표기한다.
MTM: Transmitter Gate
y는 자기 자신(y)과 활동 f(x)에 의해 조절 — 활동이 클수록 소진, 없으면 K로 회복
LTM: Learning Rule
z는 활동 f(x)와 자기 자신(z)에 의해 변화 — f(x)가 게이트, (h−z)가 오차 신호
생산율(production rate). 비활성 시 전달물질 소포(vesicle)가 재합성되는 속도. 높을수록 빠르게 K로 회복.
최대 전달물질 용량. 시냅스 소포 풀(vesicle pool)의 상한. y는 항상 [0, K] 범위에서 움직인다.
소모율(depletion rate). 활동 f(x)에 비례하여 전달물질이 소진되는 속도. Habituation의 강도를 결정.
학습률(learning rate). f(x)에 의해 게이트되어, 활동이 없으면 학습 정지. Gated steepest descent의 스텝 크기.
전체 네트워크에서의 표기 (첨자 j→i 포함)
MTM
각 시냅스 j→i마다 독립된 전달물질 yji. 사전 세포 j의 활동 fj(xj)가 클수록 해당 시냅스의 전달물질만 소진된다.
LTM
각 시냅스 j→i마다 독립된 가중치 zji와 목표 hji. 여기서 x는 개별 뉴런이 아닌 네트워크 전체 상태 벡터 (x₁, x₂, …, xₙ)이다 — h는 전체 활동 패턴에 의존하여 목표를 정한다. 사전 세포 j가 활성일 때만 해당 시냅스의 가중치가 목표를 향해 이동한다.
h(x)란 무엇인가?
LTM 학습 규칙 dz/dt = Mf(x)(h−z)에서 h(x)는 "지금 이 시냅스의 가중치가 얼마여야 하는가"를 알려주는 목표 신호(target signal)이다. sigmoid 같은 활성화 함수와 전혀 다르다. 활성화 함수는 뉴런 하나의 입출력 관계를 정하지만, h(x)는 시냅스 하나의 학습 목표를 정한다.
핵심 아이디어: 학습이란 "현재 가중치 z"와 "목표 h" 사이의 오차 (h−z)를 줄여나가는 과정이다. 그런데 이 목표는 네트워크에 어떤 입력 x가 들어오느냐에 따라 달라진다 — 그래서 h(x), x의 함수이다. 같은 시냅스라도 입력 패턴 A가 들어올 때와 패턴 B가 들어올 때 목표가 다를 수 있다.
함수 h 자체는 고정인가? 그렇다. h는 네트워크 구조(architecture)의 일부로, 설계 시 결정되어 학습 중에 바뀌지 않는다. 바뀌는 것은 오직 z(가중치)뿐이다. h가 정하는 것은 "규칙" — 어떤 입력이 오면 어떤 값을 목표로 삼아라 — 이고, z가 실제로 그 목표를 향해 이동하는 것이 학습이다.
h의 형태가 학습의 성격을 결정한다:
hji = xj / Σxk
Instar 학습. 전체 입력 중 세포 j의 비율이 목표. 학습 후 가중치 벡터가 입력 패턴의 정규화된 형태를 저장한다. 패턴 인식의 기초.
hji = xj
유계 Hebbian. 사전 세포의 활동 크기 자체가 목표. 고전적 Hebb 규칙과 유사하나 z→h로 수렴하므로 가중치 폭발이 없다.
hji = 상수 c
균일 수렴. 모든 시냅스가 같은 값 c로 수렴. 입력 패턴과 무관하게 균일한 연결 강도를 만든다.
비유: h(x)는 "선생님이 정해준 정답"이고, z는 "학생의 현재 답"이다. 선생님(h)은 문제(x)마다 다른 정답을 알려주고, 학생(z)은 틀린 만큼(h−z) 교정한다. 단, 학생이 깨어 있을 때(f(x)>0)만 공부할 수 있다 — 이것이 gated learning이다. 아래 시뮬레이션에서 h(x) 슬라이더는 이 "정답"을 직접 바꿔보는 것이다.
▶ LTM 학습 규칙의 Steepest Descent 유도 (접기)
Step 1: 오차 함수 정의
목표 h(x)와 현재 가중치 z의 차이를 제곱 오차로 정의.
Step 2: 그래디언트 계산
E를 z에 대해 미분하면 오차 신호 (h−z)가 나타난다.
Step 3: f(x) 게이팅
사전 세포 활동 f(x)를 곱하면 "gated steepest descent": 활동이 없으면 학습도 정지.
Result: LTM 학습 규칙
Grossberg의 gated steepest descent — Hebbian과 달리 유계(bounded), 안정적.

Gated Exponential Time Average
LTM 방정식 dz/dt = M·f(x)·(h − z) 는 1차 저역통과 필터와 동일한 형태이다. 고정 윈도우가 아니라 지수 감쇠 커널로 과거 입력을 평균한다.
게이트 열림 (f > 0): 시간상수 τ = 1/(M·f)
→ 최근 ~τ초의 입력을 지수적으로 가중 (최근일수록 큰 가중치)
게이트 닫힘 (f = 0): dz/dt = 0
→ 학습 정지, 기억 완벽 보존 — 평균 구간에서 완전 제외
패턴이 주기적으로 반복되면 z는 정상상태(steady-state)에 수렴한다:
z̄j = Σ( fp · hpj · Δtp ) / Σ( fp · Δtp )
• 첫 수 주기: 초기값(z₀=0)의 영향 큼, z̄ 미도달
• 수 주기 후: 초기값 씻겨나감, z̄에 근접
• M↑ 또는 f↑: τ↓ → 빠르게 정상상태 도달, 최근 패턴에 치중
• M↓ 또는 f↓: τ↑ → 느리게 수렴, 더 긴 역사를 반영
