메커니즘: 잡음 억제 (Noise Suppression)
On-center off-surround shunting 네트워크는 공간적으로 균일한 입력(정보 없음)을 자동으로 억제한다. 모든 셀이 동일한 입력을 받는 "균일 패턴"은 총 조명량 I가 아무리 강해도 출력 x_i = 0이 된다. 이것이 영 공간 주파수(zero spatial frequency) 잡음 억제이다. 반면, 밝고 어두운 영역이 있는 특징 있는 패턴은 살아남는다.
균일 입력 (잡음) — 억제됨
모든 θ_i = 1/n = C/(B+C)
→ x_i = (B+C)I/(A+I) · [1/n − C/(B+C)] = 0
I가 아무리 커도 출력 = 0
비균일 입력 (특징) — 보존됨
일부 θ_i > 1/n, 일부 θ_i < 1/n
→ x_i ≠ 0 for cells with θ_i ≠ C/(B+C)
패턴(에지, 바 등)은 살아남음
핵심 방정식

적응 수준 (Adaptation Level)
잡음 억제 조건: B >> C 이면 C/(B+C) ≈ 0. 반대로 B = (n−1)C 이면 C/(B+C) = 1/n. 균일 입력에서 모든 θ_i = 1/n이므로 θ_i − C/(B+C) = 0 → x_i = 0.
Weber Law: 비례 인자 (B+C)I/(A+I)는 총 조명 I에만 의존하며 반사율 θ_i에는 무관하다. x_i의 부호(밝다/어둡다)는 I와 무관하게 θ_i와 적응 수준의 관계로만 결정된다.
인터랙티브 시뮬레이션
패턴을 선택하고 파라미터를 조정하라. 균일(Uniform) 패턴에서 x_i → 0이 되는 것을 확인하라. B = (n−1)C 조건이 만족되면 잡음 억제가 완벽해진다.

입력 패턴 I_i (각 셀의 입력 강도)
반사율 θ_i = I_i / I — 적응 수준과 비교

I 슬라이더를 움직여도 비균일 패턴의 부호(밝다/어둡다)가 바뀌지 않는다. 단, 균일 패턴은 모든 I에서 항상 0이다.
핵심 요약
잡음 억제(Noise Suppression) 원리:
• 균일 입력(θ_i = 1/n for all i) → x_i = 0 for all i, I와 무관 (잡음 억제)
• 특징 입력(some θ_i ≠ 1/n) → x_i ≠ 0 for those cells (신호 보존)
• B = (n−1)C 이면 C/(B+C) = 1/n: 완벽한 잡음 억제 조건

Weber Law (Zeiler 1963 Adaptation Level Theory):
• x_i*의 크기는 (B+C)I/(A+I)에 비례하지만, 부호는 θ_i와 C/(B+C)의 관계로만 결정
• 총 조명 I가 달라져도 "밝다/어둡다" 판단(부호)은 변하지 않는다
• 활동 범위는 [−C, 0, x_i(t), B]로 자동 제한됨
