메커니즘: 협력-경쟁 동역학 (Cooperative-Competitive Dynamics)
On-center off-surround 수용야 구조가 고양이 망막(Kuffler 1953), 투구게 망막(Hartline & Ratcliff 1956), 도루뱀 망막(Werblin & Dowling 1969)에서 잇달아 발견되었다. 각 발견은 동일한 생물학적 원리를 다른 수준에서 드러낸다: center에서의 흥분(협력)과 surround에서의 억제(경쟁)가 결합하여 shunting 네트워크의 생물학적 구현을 이룬다.
세 가지 역사적 발견
Cat Retina
Kuffler
1953
고양이 망막 신경절세포에서 on-center off-surround 수용야 최초 발견. 좁은 중심(+)은 빛에 흥분, 넓은 주변(-)은 빛에 억제.
Limulus Retina
Hartline & Ratcliff
1956 / 1957+
투구게(Limulus) 망막에서 subtractive lateral inhibition 발견. 인접 광수용체가 서로를 억제한다. 최초의 수학적 모델이지만 덧셈/뺄셈 방식(additive)으로, 활동이 음수로 갈 수 있고 자동 이득 제어가 없다.
Mudpuppy (Necturus) Retina
Werblin & Dowling
1969+
도루뱀 망막에서 shunting (silent) inhibition 발견. 수평세포(H)가 양극세포(B)를 곱셈/나눗셈 방식으로 억제하여 자동 이득 제어(automatic gain control)를 구현한다. Grossberg의 shunting 방정식과 정확히 일치.
도루뱀(Mudpuppy) 망막 회로
R = Receptor (광수용체), H = Horizontal cell (수평세포, 붉은색), B = Bipolar cell (양극세포). R이 빛을 받아 B를 직접 흥분(on-center)시키고, 여러 R이 H를 통해 B를 측면 억제(off-surround)한다. H→B 억제가 shunting(무음) 억제이다.
핵심: 수평세포(H)는 여러 R로부터 흥분 입력을 받아 측면으로 연결된 후 양극세포(B)를 shunting 억제한다. 이것이 on-center off-surround의 생물학적 구현이다.

망막 회로가 어떻게 Grossberg의 shunting 방정식으로 대응되는지 보여준다.
Shunting ODE — 도루뱀 회로 대응
Equilibrium
On-center (R→B, 직접 흥분)
Off-surround (R→H→B, 측면 억제)
Subtractive vs Shunting — 핵심 차이
Limulus 방식 — Subtractive (가산)
평형: x* = (I − J) / A
문제: x가 음수 가능, 자동 이득 제어 없음, 입력이 커지면 모두 포화.
Mudpuppy 방식 — Shunting (곱셈)
평형: x* = BI / (A + I + J)
장점: x ∈ [0, B] 자동 경계, 입력 비율 보존(ratio preservation), 자동 이득 제어.
인터랙티브 시뮬레이션 — Subtractive vs Shunting 비교
1D 세포 배열에 공간 패턴을 적용하여 두 모델의 출력을 비교한다. 입력을 바꾸거나 파라미터를 조정해 shunting의 이점을 직접 확인하라.

Input Pattern I[i] — 공간 입력

Shunting (Mudpuppy) — x* = B·I / (A + I + J)
핵심 요약
도루뱀 망막의 수평세포(H)→양극세포(B) 억제는 shunting(무음) 억제이다.
Limulus의 감산적 억제와 달리, shunting 억제는 자동 이득 제어를 제공하고 활동을 [0, B] 범위에 묶는다. 입력이 아무리 커져도 비율이 보존되고 음수 활동이 없다.

• Werblin & Dowling (1969): Shunting inhibition — 올바른 생물학적 메커니즘
