베버의 법칙과 적응 수준 — Hyperpolarization vs Silent Inhibition
완전한 션팅 방정식(흥분 채널 + 억제 채널)의 평형을 풀면 활동 x_i*가 세 요소의 곱으로 분해된다. 과분극 억제(C > 0)가 있어야만 적응 수준 C/(B+C)가 0보다 커지고, 이를 기준으로 "밝다" (x > 0) / "어둡다" (x < 0)를 부호화할 수 있다. C = 0(무음 억제)이면 모든 활동이 항상 ≥ 0이어서 "평균보다 어두운" 대비를 표현할 수 없다.

Weber Law
자동 이득 제어. I가 아무리 커져도 B+C로 포화. 분모의 I가 이득을 자동 조절 → 베버의 법칙 (JND ∝ I)

Adaptation Level
내부 기준점. θ_i > C/(B+C)이면 양(밝다), θ_i < C/(B+C)이면 음(어둡다). C=0이면 기준점=0

Silent vs Hyperpolarizing Inhibition 비교
동일한 반사율 패턴 θ = [0.70, 0.50, 0.30, 0.20, 0.10]에 두 모델을 적용한다. C = 0(무음)이면 모두 ≥ 0이어서 "어두운" 셀을 표현할 수 없다. 현재 슬라이더의 C 값이 오른쪽 그래프에 반영된다.
C = 0 (Silent Inhibition)
x_i = B·I·θ_i / (A+I) ≥ 0, 적응 수준 없음
C > 0 (Hyperpolarizing Inhibition)
x_i = (B+C)I/(A+I) · [θ_i − C/(B+C)]
C 슬라이더를 0으로 내리면 두 그래프가 같아지고, C > 0이면 기준점 이하 셀이 음수가 됩니다.
Weber 이득 (B+C)·I / (A+I) — I 증가에 따른 포화
조명 불변성 데모 — I를 스케일해도 패턴 부호 보존
조명 불변성 (Illumination Invariance) 인터랙티브
조명 배율을 바꾸면 I가 일괄 스케일된다. θ_i = I_i/I는 변하지 않으므로, 활동의 부호와 상대 패턴이 보존된다. Weber 이득 (B+C)I/(A+I)는 전체 크기만 바꾼다.
50×
조명이 50배 증가해도 패턴의 부호(밝다/어둡다)와 상대 순위는 변하지 않습니다.
핵심: C = 0(무음 억제)이면 적응 수준 = 0이어서 모든 활동이 ≥ 0이다. C > 0(과분극 억제)이면 적응 수준 C/(B+C)가 생겨, 이보다 약한 입력은 음의 활동(어둡다)으로, 강한 입력은 양의 활동(밝다)으로 부호화된다. 이것이 신경 수준에서 '밝기 대비'가 표현되는 메커니즘이다.
