## 개요: STM과 LTM의 분리

### HM 환자 사례 (Scoville와 Milner, 1957)

**핵심 발견**: STM과 LTM의 특성은 분리될 수 있다.

1957년, William Scoville와 Brenda Milner는 뇌의 해마(hippocampus)와 인근 구조를 양측적으로 제거한 HM 환자에 대한 획기적인 연구를 발표했습니다. 이 수술은 27세의 Henry Molaison(HM이라는 가명으로 알려짐)의 통제 불가능한 간질을 치료하기 위해 시행되었습니다.

**수술의 결과**:
- **인지 기능**: Henry의 많은 인지 기능은 수술 후에도 보존되었습니다
- **기억 결손**: 새로운 명시적 의식적 기억(explicit conscious memories)의 장기 기억 저장에 심각한 결함이 발생했습니다
- **STM 유지**: Henry는 최근 사건들을 STM에 저장할 수 있었지만
- **주의 산만**: 만약 주의가 산만해지면, 사건들의 모든 흔적이 사라졌습니다

**대표적 예시**:
- Henry에게 소개를 받으면 따뜻한 환영을 보내지만
- 방을 나갔다가 다시 들어오면 다시 소개받음
- 이전 만남에 대한 기억이 없음

### 프로시저럴 기억의 보존

흥미롭게도, Henry의 **프로시저럴 기억(procedural memory)** 는 온전히 보존되었습니다:
- 운동 기술 학습 및 기억 능력 유지
- 기술 연습에 대한 의식적 기억이 없음에도 불구하고 기술 습득 가능

이러한 선천성과 후천성 기억의 다른 특성은 뇌의 **계산적 보완 피질 처리 스트림**을 보여줍니다. Table 1.1에서 설명된 것처럼, 선천성과 후천성 기억의 다른 특성들은 뇌의 조직화 방식을 설명합니다.

## STM과 LTM의 상호작용 문제

### 역사적 맥락

1957년, Grossberg가 자신의 STM-LTM 모델에 대한 법칙을 파생시킬 때, STM과 LTM 사이의 강력한 상호작용으로 인해 이 두 과정이 혼합되거나 함께 묶일 수 있다는 불확실성이 존재했습니다. 바로 같은 해, Scoville와 Milner의 연구가 발표되었습니다.

### 학습의 기능적 단위

1960년대 Grossberg의 네트워크 개념적 및 수학적 분석은 학습의 기능적 단위가 개별 세포 쌍 간의 흔적이 아닌, 네트워크 전반에 걸쳐 분산된 **패턴(patterns) 또는 벡터(vectors)** 의 LTM 흔적이라는 것을 보여주었습니다.

**시리얼 러닝 데이터 분석**:
- 분배된 LTM 패턴이 STM 패턴과 일치하기 위해 어떻게 학습하는지 이해
- 구둣곡선 형태의 시리얼 포지션 곡선(serial position curve) 설명
- 어떤 LTM 흔트는 증가해야 하고, 다른 것들은 감소해야 함

### LTM 법칙의 이중 성격

LTM 법칙은 나중에 과학자들이 **헵시안(Hebbian)** 과 **반-헵시안(anti-Hebbian)** 성질이라고 부를 것을 결합했습니다:

- 증가: LTM 흔트가 큰 STM 활동과 일치할 때 (헵시안)
- 감소: LTM 흔트가 작은 STM 활동과 일치할 때 (반-헵시안)

**수학적 표현**:
- LTM 흔트 $z_{AB}$는 후신경 STM 신호 $x_B$ 값을 추적하고 접근
- 방향: **가장 급강하(steepest descent)**

## 헵스 가설의 한계

### 헵스의 기본 가설 (1949)

헥스는 1949년 다음과 같이 기술했습니다:
> "세포 A의 축상(axon)이 세포 B를 흥분할 만큼 가까이에 있을 때... B를 발화시키는 세포들 중 하나로서 A의 효능은 증가한다"

헥스 가설의 문제점:

1. **잘못된 계산 단위 선택**:
   - 헥스는 단일 경로를 계산 단위로 선택
   - 실제 계산 단위는 네트워크 전반에 걸쳐 분산된 LTM 패턴

2. **유한한 상한 문제**:
   - LTM 흔트가 상한값에 도달하면
   - 추가 학습이 불가능
   - 세상이 변해도 적응 불가능

### 게이트드 가장 급강하 학습

**개념**: 학습은 전신경 STM "샘플링" 신호가 충분히 활성화될 때만 발생

**수학적 표현**:
$$\frac{dz_{ki}}{dt} = \begin{cases}
0 & \text{if } f_k^{sample}(x) \leq 0 \\
-f_k^{sample}(x) [h_i(x) - z_{ki}] & \text{if } f_k^{sample}(x) > 0
\end{cases}$$

**작동 원리**:
- **게이트(장벽)**: $f_k^{sample}(x) > 0$일 때만 학습 가능
- **가장 급강하**: $h_i(x) - z_{ki}$를 따라 LTM 흔트가 STM 신호 값에 수렴
- **비대칭**: LTM 흔트가 증가하거나 감소할 수 있음

**실험적 확인**:
- 20년 후, Wolf Singer와 William Levy가 신경생리학적으로 확인
- 뇌의 LTM 흔트가 게이트드 가장 급강하 특성을 보임

## 샌팅 모델 (Shunting Model)의 도출

### 문제의 제기

Grossberg는 막 방정식(membrane equations)을 모르는 상태에서 순수한 사고 실험을 통해 샌팅 모델을 도출했습니다. 이 모델은:

1. **자동 이득 제어(automatic gain control, AGC)**
2. **비율(패턴) 보존**
3. **무한 동적 범위 처리**

### 막 방정식과 샌팅 모델의 일치

막 방정식:
$$C\frac{\partial V}{\partial t} = (V^+ - V)g^+ + (V^- - V)g^- + (V^p - V)g^p$$

- 각 항은 "전압 차이 × 전도도" 형태
- 곱셉적(shunting) 구조, 덧셈적이 아님
- 생물물리학적 구조와 일치

### Additive Model vs Shunting Model

**Additive Model**:
$$\frac{dx_i}{dt} = -A_i x_i + \sum_j f_j(x_j) z_{ji} + I_i$$

- 모든 영향력이 단순히 더해짐
- 활동 $x_i$에 상한/하한이 없음

**Shunting Model**:
$$\frac{dx_i}{dt} = -A_i x_i + (B_i - C_i x_i)[\cdots] - (E_i x_i + F_i)[\cdots]$$

- 흥분 입력이 $(B_i - C_i x_i)$와 곱해짐
- 억제 입력이 $(E_i x_i + F_i)$와 곱해짐
- 활동이 자동으로 $[-F_i/E_i, B_i/C_i]$ 범위에 묶임

## MTM (Medium-Term Memory)과 습관화 게이트

### MTM과 LTM의 시간적 차이

- **STM**: 가장 빠른 변화
- **MTM**: STM보다 더 느리게 변화
- **LTM**: MTM보다 더 느리게 변화

### MTM의 수학적 표현

$$\frac{dy_{kj}}{dt} = -D_{kj} y_{kj} + D_{kj} f_j(x_j) z_{kj}$$

**전달자 게이트(transmitter gate)**:
- $f_j(x_j) z_{kj}$에 의해 활성화된 신호를 곱함
- **습관적(habituative)**: 신호의 방출 또는 비활성화로 신호 경로의 효과 감소

### LTM의 수학적 표현

$$\frac{dz_{ki}}{dt} = \begin{cases}
0 & \text{if } f_k^{sample}(x) \leq 0 \\
-f_k^{sample}(x) [h_i(x) - z_{ki}] & \text{if } f_k^{sample}(x) > 0
\end{cases}$$

**학습 게이트**:
- 학습 게이트가 열릴 때만 LTM 조정 가능
- LTM이 학습 내용을 유지 until 게이트가 다시 열림

### 수학적 구조의 중요성

이 모든 방정식들은:
- 뇌에서 발견되는 다양한 데이터를 설명하고 예측
- 신경망의 STM과 LTM 상호작용이 심리적 및 신생물학적 데이터를 설명하고 예측
- 적은 수의 기본 방정식이 중요하고 흥미로운 데이터를 설명할 수 있음

## 세 가지 신경망 연구의 근원

### 1. 이진 모델 (Binary Models)

**McCulloch-Pitts 모델** (1943):
- 이진 시스템
- 뉴런 신호들이 모두 또는 없는 스파이크로 전달
- 활동이 발화 임계값을 초과하면 스파이크 생성
- 시그넘 함수(sgn) 사용: +1 (양수 활동), 0 (제로), -1 (음수 활동)

**영향**: John Von Neumann의 디지털 컴퓨터 개발에 큰 영향

### 2. 선형 모델 (Linear Models)

**Perceptron** (Rosenblatt, 1958):
- 연속 시간에서 진화
- 활동이 자연스럽게 감쇠
- 비제로 임계값에서 이진 신호 생성

**학습 문제**:
- STM과 LTM 상호작용이 분리되어 분석 단순화
- 먼저 STM 변화, 그다음 LTM 조정, 반복
- 자율적인 실시간 작동 불가능

### 3. 연속 비선형 모델 (Continuous and Nonlinear)

Grossberg의 작업은 주로 이 분야에 기여했습니다:
- 비순환(feedforward)과 순환(recurrent) 네트워크
- 실시간 자율 작동
- STM과 LTM이 변화하는 환경에 동시에 적응

## 결론: 학습 이해의 진보

Grossberg의 1957-58년 Dartmouth 대학 1학년 때의 수학적 분석은:

1. **STM과 LTM 상호작용의 패러다임**을 제시
2. **비선형 미분 방정식 시스템**으로 신경망 설명
3. **심리적 및 신생물학적 데이터** 설명 및 예측

이러한 초기 모델(Additive Model과 Shunting Model)은 오늘날 거의 모든 생물학적 모델러들에게 사용되고 있습니다. MTM과 LTM 법칙은 이러한 STM 방정식과 함께 작동하여 인간과 동물에서 관찰되는 다양한 학습 현상을 설명합니다.
