## 1. 도입: 무한 동적 범위 문제와 자동 이득 제어

### Gedanken Experiment: 사고 실험에서 출발하기

이 장의 논의는 **Gedanken Experiment(사고 실험)** 에서 시작합니다 (Figure 2.19, Grossberg 1970). Gedanken은 독일어로 "사고(thought)"를 뜻하며, 실제 실험 없이 **순수한 수학적 추론만으로** 결론을 도출하는 방법입니다. 물리학에서 Einstein("빛의 속도로 달리면서 거울을 보면?" → 특수상대성이론)이나 Schrödinger("상자 안의 고양이" → 양자역학 해석)가 사용한 것과 같은 전통입니다. Grossberg의 사고 실험은 "유한한 뉴런으로 무한 입력 범위를 처리하려면 어떤 수학 구조가 필요한가?"를 묻습니다.

### 문제: 입력 범위가 무한하다

뇌의 뉴런은 극히 작은 신호부터 엄청나게 큰 신호까지 처리해야 합니다. 그런데 뉴런의 활동(activity)에는 물리적 상한이 있습니다. 어떻게 **무한한 입력 범위**(infinite dynamic range)를 **유한한 활동 범위** 안에 담을 수 있을까요?

### 해결: 곱셈적 제어와 자동 이득 제어

"그냥 시그모이드(로지스틱) 함수를 쓰면 무한 입력을 유한 출력에 넣을 수 있지 않은가?"라고 생각할 수 있습니다. 하지만 시그모이드는 **절대값을 압축**할 뿐, 입력이 커지면 모든 셀이 포화되어 패턴 정보(셀 간 차이)가 사라집니다. 마찬가지로 감마 보정($\text{output} = \text{input}^\gamma$)도 각 채널에 독립적으로 적용되므로 입력 간 **비율이 왜곡**됩니다.

Grossberg의 핵심 통찰은 입력이 활동을 **곱셈적으로(multiplicatively)** 제어해야 한다는 것입니다. 책의 Figure 2.9와 2.10에서 덧셈적 모델과 곱셈적(shunting) 모델을 비교합니다:

**덧셈적 모델** (Figure 2.9, Additive Model):

$$\frac{d}{dt}x_i = -A_i x_i + \sum_j f_j(x_j) z_{ji} + I_i$$

→ 흥분·억제·입력이 그냥 **더해짐**. 활동 $x_i$에 상한/하한이 없음.

**Shunting 모델** (Figure 2.10, Shunting Model):

$$\frac{d}{dt}x_i = -A_i x_i + (B_i - C_i x_i)\left[\sum_j f_j(x_j)D_{ji}y_{ji}z_{ji} + I_i\right] - (E_i x_i + F_i)\left[\sum_j g_j(x_j)G_{ji}Y_{ji}Z_{ji} + J_i\right]$$

→ 흥분 입력이 $(B_i - C_i x_i)$와 **곱해지고**, 억제 입력이 $(E_i x_i + F_i)$와 **곱해짐**. 이 곱셈 구조 덕분에 활동이 자동으로 $[-F_i/E_i, \; B_i/C_i]$ 범위에 묶이며, 현재 활동 수준이 입력의 효과를 자동 조절합니다:

- $x_i$가 작을 때 → $(B_i - C_i x_i)$가 큼 → 흥분 입력 효과 **큼** (높은 게인)
- $x_i$가 $B_i/C_i$에 가까울 때 → $(B_i - C_i x_i)$가 작음 → 흥분 입력 효과 **작음** (낮은 게인)

위 수식이 복잡해 보이지만, feedforward on-center off-surround 네트워크의 **특수한 경우**로 단순화하면 직관이 명확해집니다. $C_i = 1$, $B_i = B$, $E_i = 1$, $F_i = 0$으로 놓고, 흥분 입력을 $I_i$ (on-center), 억제 입력을 $\sum_{k \neq i} I_k$ (off-surround)로 잡으면:

**단순화된 shunting 방정식:**

$$\frac{d}{dt}x_i = -Ax_i + (B - x_i)I_i - x_i \sum_{k \neq i} I_k$$

**단순화된 additive 방정식** (비교용):

$$\frac{d}{dt}x_i = -Ax_i + I_i - \sum_{k \neq i} I_k$$

차이가 보이시나요? Shunting 버전에서는 흥분 입력 $I_i$에 $(B - x_i)$가 곱해지고, 억제 입력에 $x_i$가 곱해집니다. Additive 버전에서는 그냥 더하고 빼기만 합니다. 이 곱셈 하나의 차이가 자동 이득 제어, 비율 보존, 노이즈 억제 등 이 장의 모든 성질을 만들어냅니다.

이것이 바로 **자동 이득 제어(automatic gain control, AGC)** 입니다. AGC 자체는 라디오·오디오·카메라 등 신호처리 전반에서 흔히 쓰이는 개념이지만, shunting 네트워크의 AGC는 단순한 단일 채널 볼륨 조절과 다릅니다 — **여러 채널이 경쟁하면서 패턴(비율)을 보존**합니다:

| | 일반 AGC / 시그모이드 / 감마 | Shunting AGC |
|---|---|---|
| 적용 | 각 채널 독립 | 채널 간 경쟁 |
| 보존하는 것 | 출력 범위 (절대값) | **입력 간 비율 (패턴)** |
| 균일 입력 | 여전히 출력 있음 | 출력 = 0 (노이즈 억제) |
| 파라미터 | 고정 ($\gamma$, 포화값) | 실시간 자동 조절 |

### 왜 단순한 shunting만으로는 부족한가: 포화 문제 (Figures 2.19-2.21)

Shunting 모델이 곱셈적이라 자동 이득 제어가 있다고 했지만, **셀 간 상호작용이 없는** 단순 shunting에는 치명적 문제가 있습니다.

**상호작용 없는 shunting** (Figure 2.20):

$$\frac{d}{dt}x_i = -Ax_i + (B - x_i)I_i$$

**평형(equilibrium)** 이란 활동이 더 이상 변하지 않는 안정 상태, 즉 $\frac{d}{dt}x_i = 0$인 순간을 말합니다. 입력이 일정하게 유지되면 뉴런의 활동은 시간이 지남에 따라 어떤 값에 수렴하는데, 그 수렴 값이 평형입니다. 이 문서에서 "평형에서"라는 말은 항상 "$\frac{d}{dt}x_i = 0$으로 놓고 $x_i$에 대해 풀면"이라는 뜻입니다.

위 미분 방정식에서 $\frac{d}{dt}x_i = 0$으로 놓으면:

$$0 = -Ax_i + (B - x_i)I_i = -Ax_i + BI_i - x_i I_i$$

$$0 = BI_i - x_i(A + I_i)$$

$$x_i = \frac{BI_i}{A + I_i} = \frac{B\theta_i I}{A + \theta_i I}$$

여기서 $I_i = \theta_i I$ ($\theta_i$: 상대 강도/반사율, $I$: 총 강도/조도).

**문제** (Figure 2.21): 총 입력 $I$가 커지면, $A + \theta_i I \approx \theta_i I$이므로 $x_i \approx B$ — **모든 셀이 상한 $B$에 포화**됩니다. 패턴 정보($\theta_i$ 간의 차이)가 사라집니다!

- $I$가 작으면 → 노이즈에 묻힘
- $I$가 크면 → 전부 포화

이것이 바로 **noise-saturation 딜레마**입니다 (Figure 2.19). 생물학에서 무한은 존재하지 않으므로, 유한한 활동 범위 안에서 패턴을 보존해야 합니다.

### 해결: On-center off-surround 네트워크 (Figures 2.22-2.23)

패턴에 민감한 비율 $\theta_i = I_i / \sum_j I_j$를 계산하려면 어떤 상호작용이 필요할까요? (Figure 2.22)

- $I_i$가 증가하면 → $\theta_i$가 증가 → **흥분(excitation)** 이 필요
- $I_k$ ($k \neq i$)가 증가하면 → $\theta_i$가 감소 → **억제(inhibition)** 가 필요

즉, 자기 자신의 입력은 흥분시키고(**on-center**), 다른 셀들의 입력은 억제하는(**off-surround**) 구조가 필요합니다.

**Shunting on-center off-surround 네트워크** (Figure 2.23):

$$\frac{d}{dt}x_i = -Ax_i + (B - x_i)I_i - x_i \sum_{k \neq i} I_k$$

평형($\frac{d}{dt}x_i = 0$)에서 $x_i$에 대해 풀면:

$$0 = -Ax_i + BI_i - x_i I_i - x_i \sum_{k \neq i} I_k = BI_i - x_i\!\left(A + I_i + \sum_{k \neq i} I_k\right)$$

$$x_i = \frac{BI_i}{A + I_i + \sum_{k \neq i} I_k} = \frac{BI_i}{A + I} = \theta_i \cdot \frac{BI}{A + I}$$

여기서 $I = \sum_k I_k$이고, 마지막 등호는 $I_i = \theta_i I$를 대입한 것입니다.

이 결과가 핵심입니다:

- $\theta_i = I_i / I$ — **비율(패턴)이 정확히 보존됨**, $I$가 아무리 커져도
- $\frac{BI}{A+I} \leq B$ — 총 활동이 $B$ 이하로 **정규화**됨
- $I \to \infty$여도 $x_i \to B\theta_i$ — **포화 없이** 비율 유지 (상호작용 없는 경우의 $x_i \to B$와 대조)

총 활동의 합:

$$x = \sum_{k=1}^{n} x_k = \frac{BI}{A + I} \leq B$$

> **비유**: 여러 수도꼭지가 하나의 수도관에 연결되어 있다고 상상하세요. 수도관의 총 수압(총 활동)에는 한계가 있습니다. 한 꼭지를 크게 열면(강한 입력), 그 꼭지로 물이 많이 흐르는 대신 다른 꼭지들의 유량이 줄어듭니다. 수압이 올라가도(총 입력 증가) 각 꼭지의 **유량 비율**은 열린 정도에 따라 유지됩니다. 이것이 실시간으로, 자동으로 일어납니다.

Figure 2.23이 요약하는 이 네트워크의 성질들:

| 성질 | 의미 |
|------|------|
| No saturation | 포화 없이 무한 동적 범위 처리 |
| Automatic gain control | 자동 이득 제어 |
| Compute ratio scale | 비율(반사율) 계산 |
| Weber law | 배경 변화에 민감도 유지 |
| Conserve total activity | 총 활동 정규화 ($\leq B$) |
| Limited capacity | 제한된 용량 |
| Real-time probability | 활동이 실시간 확률처럼 동작 |

### 비순환(feedforward) vs 순환(recurrent) 네트워크

둘 다 총 활동이 제한된다는 점은 같지만, **언제/어떻게** 제한되는지가 다릅니다:

- **비순환(feedforward)**: 입력 → 한 번 처리 → 출력. 총 활동이 **즉시** $\frac{BI}{A+I}$로 결정됩니다. 입력이 바뀌면 출력도 바로 바뀌는, 계산기 같은 구조입니다 (Figure 2.23).
- **순환(recurrent/feedback)**: 셀들이 서로 피드백합니다. 총 활동이 **시간에 따라 수렴**하며, 이 수렴 과정 자체가 중요한 계산을 수행합니다:
  - **경쟁이 시간에 걸쳐 진행** → 강한 셀이 약한 셀을 점점 억제 → winner-take-all 가능
  - **패턴을 단기 기억(STM)에 저장** — 입력이 사라져도 활동 패턴이 유지됨 (Figure 1.6-1.9)
  - 활동이 **실시간 확률**처럼 동작 — 동적으로 변하면서도 합이 일정

Grossberg가 이 구분을 강조하는 이유는, 이 책에서 다루는 대부분의 아키텍처(ART, 단기 기억, 주의 등)가 **순환** 네트워크이기 때문입니다. 비순환은 개념 설명의 출발점이고, 순환이 실제 뇌 모델입니다.

## 2. 신경생리학의 막 방정식 (Figure 2.24)

> **"Deriving the membrane equations of neurophysiology from a thought experiment."**
> (사고 실험으로부터 신경생리학의 막 방정식을 도출하기)

앞서 다룬 shunting(질량 작용) 역학이 실제로 **신경생리학의 막 방정식(membrane equations)** 과 동일하다는 것이 이 절의 핵심입니다. Grossberg는 순수한 사고 실험 — noise-saturation 딜레마를 해결하려는 수학적 추론 — 에서 출발했는데, 그 결과로 얻어진 방정식이 이미 신경생리학에서 실험적으로 확립된 막 방정식과 정확히 같은 형태임을 보여줍니다. 이론과 실험이 독립적으로 같은 결론에 도달한 것입니다.

<img src="images/fig2_24.jpeg" alt="Figure 2.24 — 신경생리학의 막 방정식" width="50%">

### 핵심 수식: 막 방정식

$$C\frac{\partial V}{\partial t} = (V^+ - V)g^+ + (V^- - V)g^- + (V^p - V)g^p$$

#### 변수 설명

| 기호 | 의미 | 역할 |
|------|------|------|
| $V$ | 세포막 전압 (voltage) | 뉴런의 현재 활동 수준 |
| $C$ | 막 커패시턴스 (capacitance) | 전압 변화의 시간 상수 결정 |
| $V^+$ | 흥분성 포화 전압 (excitatory saturation voltage) | 전압이 올라갈 수 있는 최대값 |
| $V^-$ | 억제성 포화 전압 (inhibitory saturation voltage) | 전압이 내려갈 수 있는 최소값 |
| $V^p$ | 수동 포화 전압 (passive saturation voltage) | 입력 없을 때의 안정 전압 |
| $g^+$ | 흥분성 전도도 (excitatory conductance) | 흥분 채널의 "열린 정도" |
| $g^-$ | 억제성 전도도 (inhibitory conductance) | 억제 채널의 "열린 정도" |
| $g^p$ | 수동 전도도 (passive conductance) | 누출 채널의 "열린 정도" |

#### 직관적 이해

이 방정식의 각 항은 **"전압 차이 × 전도도"** 형태입니다:
- $(V^+ - V)g^+$: 현재 전압이 $V^+$보다 낮으면 위로 끌어올림 (흥분)
- $(V^- - V)g^-$: 현재 전압이 $V^-$보다 높으면 아래로 끌어내림 (억제)
- $(V^p - V)g^p$: 현재 전압을 수동 안정점으로 되돌림 (누출)

전도도($g$)가 **입력에 의해 조절**되므로, 이 방정식은 본질적으로 **곱셈적(shunting)** 입니다. 단순한 덧셈(additive)이 아닙니다.

> **용어 참고 — "shunting"의 어원**: 전기공학에서 shunt는 **병렬로 연결된 우회 경로(분류기)** 를 뜻합니다. 막 방정식의 등가 회로에서 흥분·억제·수동 이온 채널이 세포막에 **병렬로** 연결되어 있고, 각 채널(전도도 $g$)이 전류를 자기 쪽으로 **분류(shunt)** 합니다. 같은 수학 구조 $(B-V) \cdot I$를 가리키는 세 가지 이름이 있습니다:
> - **Shunting** — 병렬 전도도를 통한 전류 분류 (회로적 관점)
> - **Multiplicative** — 입력 × 상태변수의 곱 (수학적 관점)
> - **Mass action (질량 작용)** — 반응물 농도의 곱에 비례하는 반응 속도 (화학 동역학 관점)

### Feedforward shunting on-center off-surround 네트워크로의 단순화

Figure 2.24 우측 하단에서, 특수한 값을 대입합니다:

| 설정 | 값 |
|------|-----|
| $V^+ = B$ | 흥분 포화 전압 |
| $C = 1$ | 커패시턴스 정규화 |
| $V^- = V^p = 0$ | 억제·수동 포화 전압 = 0 (**침묵 억제**) |
| $g^+ = I_i$ | on-center 입력 |
| $g^- = \sum_{k \neq i} I_k$ | off-surround 입력 |

이 대입을 통해 막 방정식은 곧바로 **feedforward shunting on-center off-surround 네트워크** 방정식이 됩니다. 즉, 사고 실험에서 도출한 수학적 구조가 실제 신경 세포의 물리 법칙과 **정확히 일치**합니다.

> **핵심 메시지**: 막 방정식과 on-center off-surround 해부학은 진화 과정에서 **공동 진화(co-evolve)** 했을 가능성이 높습니다. 이는 세포 수준의 가장 초기 단계부터 분산 패턴 처리를 가능하게 한 보편적 발달 코드(universal developmental code)의 일부입니다.

## 3. Weber 법칙, 적응, 이동 특성 (Figures 2.25-2.26)

> **"Weber law, adaptation, and shift property."**
> (Weber 법칙, 적응, 그리고 이동 특성)

Grossberg는 가장 단순한 feedforward shunting on-center off-surround 네트워크에도 뇌에서 발견되는 중요한 성질들이 이미 내재해 있음을 보여줍니다. 그 중 하나가 **이동 특성(shift property)** 이며, 이를 통해 네트워크가 입력 범위의 변화에도 민감도를 유지하는 원리를 설명합니다.

> **용어 참고 — Weber 법칙 vs Fechner 법칙**: 심리물리학에서 **Weber 법칙**은 "최소 식별 차이(JND)가 자극 강도에 비례한다"($\Delta I / I = \text{const}$)는 것이고, **Fechner(Weber-Fechner) 법칙**은 이를 적분하여 "감각 = $k \cdot \ln(I)$"로 대수적 관계를 도출한 것입니다. Grossberg가 여기서 "Weber 법칙"이라 부르는 것은 더 넓은 의미로, on-center off-surround 네트워크가 **on-center 입력의 민감도를 off-surround 총 입력으로 나눗셈적으로 조절**하는 성질을 가리킵니다. 대수 좌표에서의 이동 특성이 이 나눗셈적 조절의 직접적 결과입니다.

### On-center off-surround 평형에서 Weber 법칙으로

이 절의 수식은 갑자기 등장하는 것이 아니라, 앞 절(Figure 2.23)의 **feedforward shunting on-center off-surround 네트워크 평형 활동**에서 직접 출발합니다:

$$x_i = \frac{BI_i}{A + I}$$

여기서 $I = I_i + \sum_{k \neq i} I_k$, 즉 on-center 입력($I_i$)과 off-surround 입력($J = \sum_{k \neq i} I_k$)의 합입니다. 이를 $I_i$와 $J$로 분리하면:

$$x_i = \frac{BI_i}{A + I_i + J}$$

이 수식의 구조를 더 잘 보기 위해, 대수 좌표로 변환합니다:

$$K = \ln(I_i), \quad I_i = e^K, \quad J = \sum_{k \neq i} I_k$$

- $K$: on-center 입력의 로그값 (가로축)
- $J$: off-surround 총 입력 (배경 조명에 해당)

<img src="images/fig2_25.png" alt="Figure 2.25 — Weber 법칙, 적응, 이동 특성" width="50%">

### 대수 좌표에서의 평형 활동

$$x_i(K, J) = \frac{BI_i}{A + I_i + J} = \frac{Be^K}{A + e^K + J}$$

이 수식에서 off-surround 총 입력 $J$가 분모에 있으므로, $J$가 커지면 같은 on-center 입력 $I_i$에 대한 반응이 줄어듭니다 — **나눗셈적 정규화**입니다. 이것이 바로 Grossberg가 "Weber 법칙"이라 부르는 성질의 수학적 실체입니다.

### 이동 특성 (Shift Property) — Weber 법칙의 핵심

$$x(K + S, J_1) = x(K, J_2), \quad S = \ln\left(\frac{A + J_1}{A + J_2}\right)$$

이 수식이 말하는 것:

> **배경($J$)이 바뀌면, 활동 곡선이 $K$ 축을 따라 수평 이동할 뿐, 형태나 최대값은 변하지 않는다.**

이것이 바로 **Weber 법칙**입니다. off-surround가 on-center 입력의 효과를 **나눗셈적으로** 조절하기 때문에 발생합니다.

- $J$가 커지면 ($J_1 < J_2$로) → 곡선이 오른쪽으로 이동 (더 큰 입력이 필요)
- $J$가 작아지면 → 곡선이 왼쪽으로 이동 (작은 입력에도 반응)
- **압축(compression)이 일어나지 않음**: 최대 활동값 $B$는 $J$에 무관

> **일상적 비유**: 어두운 방에서는 작은 촛불도 밝게 느껴지고, 밝은 낮에는 같은 촛불이 보이지 않습니다. 하지만 밝은 낮에도 태양 대비 다른 물체의 **상대적** 밝기 차이는 여전히 감지할 수 있습니다. 이것이 이동 특성입니다.

### Mudpuppy 망막에서의 실험적 검증 (Figure 2.26)

> **"Shift property and shunting on-center off-surround network in the mudpuppy retina."**
> (Mudpuppy 망막에서의 이동 특성과 shunting on-center off-surround 네트워크)

이론적으로 예측된 이동 특성이 실제 생물학적 망막에서도 관찰되는지를 검증한 핵심 실험입니다. 중요한 것은 일어나지 *않은* 일입니다 — 세포 반응의 동적 범위가 줄어들지(압축되지) 않았다는 것입니다.

<img src="images/fig2_26.jpeg" alt="Figure 2.26 — Mudpuppy 망막의 이동 특성" width="50%">

1971년 Frank Werblin은 수생 도롱뇽인 **Mudpuppy**(*Necturus maculosus*)의 망막에서 이 이동 특성을 실험적으로 관찰했습니다 (Werblin, 1971).

"압축이 일어나지 않았다"는 것만으로는 shunting 모델의 근거가 약합니다. 다른 메커니즘도 비슷한 결과를 낼 수 있기 때문입니다. Figure 2.26은 실제로 **여러 대안 가설을 동시에 검증**합니다:

| 관찰 사실 | 배제되는 대안 | 이유 |
|----------|------------|------|
| **(a) 상대적 figure-to-ground 계산** | 절대값 기반 모델 (시그모이드, 감마) | 각 셀 독립 처리로는 상대 대비 계산 불가 |
| **(b) 반응이 $I(A+J)^{-1}$ 형태** | Fechner 대수 압축 ($k \cdot \ln I$) | Fechner는 배경 $J$에 의한 나눗셈이 없음. Shunting 모델만이 $\frac{BI_i}{A+I_i+J}$ 형태를 예측 |
| **(c) 과분극 없음 (침묵 억제)** | 과분극 기반 억제 모델 | 억제가 수동 전압 이하로 내려가지 않음 → $V^- = V^p$ |
| **(d) 이동 특성 + 압축 없음** | Hartline-Ratliff 감산적 측면 억제 | 감산 모델은 배경이 커지면 반응 범위가 **압축됨**. Shunting은 이동만 함 |

또한 이 데이터가 설득력을 가지는 이유는, 이전 연구(Werblin, Dowling et al. 1969+)에서 mudpuppy 망막의 **해부학적 on-center off-surround 구조**(R→B 흥분, R→H→B 억제)가 이미 **독립적으로** 확인되었기 때문입니다. 즉:
- **이론**: shunting on-center off-surround 모델이 이동 특성을 예측
- **해부학**: 해당 구조가 실제로 존재함을 독립 확인
- **생리학**: 예측된 반응 패턴(이동, 무압축, 침묵 억제)이 관찰됨

이 세 가지의 수렴이 단순한 "일어나지 않은 일" 이상의 근거를 제공합니다.

## 4. 협력-경쟁 역학과 Mudpuppy 망막 (Figure 2.27)

### 역사적 맥락

| 연구자 | 연도 | 발견 |
|--------|------|------|
| Kuffler | 1953 | 고양이 망막에서 on-center off-surround 수용장 발견 |
| Hartline & Ratliff | 1956/57+ | 투구게(limulus) 망막에서 **감산적(subtractive)** 측면 억제 발견 |
| Werblin, Dowling et al. | 1969+ | Mudpuppy 망막에서 **shunting** 협력-경쟁 역학 발견 |

### Mudpuppy 망막의 구조 (Figure 2.27)

<img src="images/fig2_27.jpeg" alt="Figure 2.27 — 협력-경쟁 역학: Mudpuppy 망막의 on-center off-surround 네트워크" width="50%">

Mudpuppy 망막에는 세 가지 핵심 세포 유형이 있습니다:

1. **수용체(R, Receptor)**: 빛을 감지하는 광수용체
2. **수평 세포(H, Horizontal cell)**: 인접 광수용체들로부터 신호를 모아 off-surround 억제 신호를 전달
3. **양극 세포(B, Bipolar cell)**: on-center 흥분 신호를 받는 세포

신호 흐름:
- **R → B** (흥분, on-center): 광수용체가 양극 세포를 직접 흥분
- **R → H → B** (억제, off-surround): 광수용체 → 수평 세포 → 양극 세포로 이어지는 억제 경로

> 이 구조 덕분에 양극 세포 층에 도달할 때쯤이면 이미 noise-saturation 딜레마가 해결되어 있습니다.

### 세포내 적응 (Intracellular Adaptation): 왜 여기서 이 이야기가 나오는가?

> **"Intracellular adaptation within photoreceptors creates a Weber law acting in time."**
> (광수용체 내의 세포내 적응은 시간 속에서 작용하는 Weber 법칙을 만들어낸다)

위에서 mudpuppy 망막의 on-center off-surround 구조(R→B→H)가 noise-saturation 딜레마를 해결한다는 것을 보았습니다. 그런데 여기서 **논리적 문제**가 생깁니다: 광수용체(R)가 양극 세포(B)와 수평 세포(H)에 신호를 보내려면, 광수용체 **자체가** 먼저 포화되지 않아야 합니다. 만약 밝은 햇빛에서 광수용체가 이미 포화되어 버리면, 아무리 좋은 on-center off-surround 네트워크가 뒤에 있어도 소용이 없습니다 — 이미 정보가 사라진 후이기 때문입니다.

그래서 Grossberg는 on-center off-surround(**세포간, 공간적**) 적응과 별개로, 광수용체 **내부**에서도 독립적인 적응이 일어나야 한다고 지적합니다. 이것은 **세포내(intracellular)** 과정으로, 공간이 아닌 **시간**을 통해 작동하는 또 다른 종류의 Weber 법칙입니다.

광수용체 **내부**에서도 적응이 일어납니다:
- 습관화 전달체 게이트(habituative transmitter gate)를 사용
- 현재 입력을 과거 입력의 시간 평균으로 **나눔**
- 밝은 조명 → 높은 시간 평균 → 반응 정규화
- 이는 **세포내(intracellular)** 과정으로, 공간이 아닌 **시간**을 통해 적응
- On-center off-surround 네트워크의 **세포간(intercellular)** 적응과 함께 작동

## 5. 침묵 억제에서 과분극과 적응 수준으로 (Figures 2.28-2.29)

> **"From silent inhibition to hyperpolarization and adaptation levels."**
> (침묵 억제에서 과분극과 적응 수준으로)

이동 특성은 가장 단순한 shunting on-center off-surround 네트워크의 성질이었습니다. Grossberg는 이 네트워크를 **단순한 단계적 정교화(simple stages of refinement)** 를 통해 발전시킵니다. 첫 번째 정교화는 "침묵 억제"를 "과분극"으로 대체하는 것이며, 이를 통해 **적응 수준(adaptation level)** 이라는 중요한 개념이 등장합니다.

### 모델 계층 구조: 제약 완화의 단계

막 방정식이 가장 일반적인 형태이고, 그 아래 단계들은 점점 더 강한 제약을 가합니다:

**① 막 방정식 (가장 일반적)**

$$C\frac{\partial V}{\partial t} = (V^+ - V)g^+ + (V^- - V)g^- + (V^p - V)g^p$$

↓ $V^- = V^p = 0$, $V^+ = B$, $C = 1$, $g^+ = I_i$, $g^- = \sum_{k \neq i} I_k$ 대입

**② Feedforward shunting on-center off-surround (침묵 억제)**

$$\frac{d}{dt}x_i = -Ax_i + (B - x_i)I_i - x_i \sum_{k \neq i} I_k$$

↓ $V^- < V^p$ 허용 (억제 포화 전압을 안정점 아래로)

**③ 과분극 포함 버전 (이 절의 주제)**

$$\frac{d}{dt}x_i = -Ax_i + (B - x_i)I_i - (x_i + C)\sum_{k \neq i} I_k$$

즉, 막 방정식의 특수 조건 $V^- = V^p$("억제가 안정점 이상으로 내려가지 못함")를 **풀어주면**, $x_i + C$ 항이 등장하고 이로부터 적응 수준이 나타납니다. 추상적인 모델 단계마다 생물물리학적 파라미터 하나씩을 조정하여 새로운 기능이 열리는 구조입니다.

### 침묵 억제 (Silent Inhibition)

가장 단순한 경우인 $V^- = V^p = 0$ 설정을 **침묵 억제**라 합니다.

**의미**: 억제가 아무리 강해져도, 활동($x_i$)을 수동 포화 전압(0) **이하로** 끌어내릴 수 없습니다. 억제는 활동을 **줄일** 수는 있지만, **음수로 만들지는 못합니다**.

### 수동 포화 전압이란 무엇인가?

막 방정식을 다시 봅시다:

$$C\frac{\partial V}{\partial t} = (V^+ - V)g^+ + (V^- - V)g^- + (V^p - V)g^p$$

**수동 포화 전압** $V^p$는 모든 입력이 꺼졌을 때($g^+ = g^- = 0$) 세포가 자연스럽게 안정되는 전압입니다. 이때 방정식은 $(V^p - V)g^p$만 남으므로, $V$는 $V^p$로 수렴합니다. 이것이 뉴런의 **안정 전위(resting potential)** — 아무 자극이 없을 때의 "기본값"입니다.

세 포화 전압의 크기 관계:

```
V⁺ (흥분 포화)     ──── 전압이 올라갈 수 있는 최대값
  │
  │  ← 흥분 입력이 V를 여기로 끌어올림
  │
V^p (수동 포화)     ──── 아무 입력 없을 때의 안정점 (resting)
  │
  │  ← 억제 입력이 V를 여기로 끌어내림
  │
V⁻ (억제 포화)     ──── 전압이 내려갈 수 있는 최소값
```

각 입력은 현재 전압 $V$를 자기 쪽 포화 전압으로 **끌어당기는** 역할을 합니다. 전도도($g$)가 클수록 더 강하게 끌어당깁니다.

### 침묵 억제 vs 과분극: 무엇이 다른가?

핵심 질문은 **$V^-$와 $V^p$의 관계**입니다:

**① 침묵 억제** ($V^- = V^p$):

```
V⁺ ─────────── 흥분 포화
  │
  │  ↑ 흥분이 끌어올림
  │
V^p = V⁻ ───── 수동 = 억제 포화 (같은 지점!)
```

억제가 끌어내리는 목표 지점($V^-$)이 안정점($V^p$)과 **같으므로**, 억제는 $V$를 안정점까지만 내릴 수 있습니다. 비유하면 자동차의 **엔진 브레이크** — 속도를 줄일 수는 있지만 후진시킬 수는 없습니다.

- 입력 있음 → $V$가 $V^p$와 $V^+$ 사이 어딘가
- 억제 → $V$를 $V^p$ 쪽으로 당김 (더 이상 안 내려감)
- 결과: 활동이 항상 $\geq 0$ → **양수/영(0)만 가능**, 음수 불가

**② 과분극** ($V^- < V^p$):

```
V⁺ ─────────── 흥분 포화
  │
  │  ↑ 흥분이 끌어올림
  │
V^p ──────────  수동 포화 (안정점)
  │
  │  ↓ 억제가 안정점 아래로 끌어내림!
  │
V⁻ ─────────── 억제 포화 (안정점보다 아래)
```

억제가 끌어내리는 목표($V^-$)가 안정점($V^p$)보다 **낮으므로**, 억제가 충분히 강하면 $V$를 안정점 **아래로** 끌어내릴 수 있습니다. 비유하면 **후진 기어** — 정지 상태를 넘어 반대 방향으로 움직일 수 있습니다.

- 억제 약함 → $V$가 $V^p$ 근처 (안정 상태)
- 억제 강함 → $V$가 $V^p$ 아래로 → **과분극(hyperpolarization)**
- 결과: 활동이 **음수 가능** → 양수/영/음수 모두 가능

### 왜 이 구분이 중요한가?

**침묵 억제**에서는 활동이 0 아래로 내려갈 수 없으므로, 평균보다 약한 입력과 아예 입력이 없는 경우를 **구분할 수 없습니다** — 둘 다 활동 = 0입니다.

**과분극**에서는 활동이 음수가 될 수 있으므로:
- $x_i > 0$: "이 특징은 평균보다 **강하다**" (흥분)
- $x_i = 0$: "이 특징은 평균과 **같다**" (적응 수준)
- $x_i < 0$: "이 특징은 평균보다 **약하다**" (과분극)

이 양수/영/음수 구분이 바로 **적응 수준(adaptation level)** 을 만들어내며, 이것이 다음에 나올 **노이즈 억제**와 **패턴 매칭**의 수학적 전제 조건입니다.

> **"활동이 음수"란 무슨 뜻인가?**
>
> 여기서 $x_i$는 **발화율(firing rate)** 이 아니라 **막 전압(membrane voltage)** 입니다. 발화율은 음수가 될 수 없지만, 전압은 안정점 아래로 내려갈 수 있습니다:
>
> - $x_i > 0$: 안정점보다 전압이 **높음** (탈분극) → 발화하기 쉬운 상태
> - $x_i = 0$: 안정점 그대로
> - $x_i < 0$: 안정점보다 전압이 **낮음** (과분극) → 발화하기 어려운 상태
>
> 실제 발화율 출력이 필요하다면 별도로 $[x_i]^+ = \max(x_i, 0)$ 처리를 합니다. 모델이 음수 전압을 내부적으로 유지하는 이유는, 그것이 "이 입력은 배경 평균보다 **약하다**"는 정보를 네트워크 계산에서 활용하기 위해서입니다.

### 과분극 (Hyperpolarization): 더 정교한 모델

실제 뇌에서는 억제가 활동을 안정점 아래로 끌어내릴 수 있습니다. 이를 위해 $V^- < V^p$로 설정합니다 (Figure 2.28).

<img src="images/fig2_28.jpeg" alt="Figure 2.28 — Weber 법칙과 적응 수준: 침묵 억제에서 과분극으로" width="50%">

수학적으로, 억제 포화 전압 $V^-$를 수동 포화 전압 $V^p$보다 작게 잡고, 이 차이를 $C > 0$으로 표현합니다:

$$\frac{d}{dt}x_i = -Ax_i + (B - x_i)I_i - (x_i + C)\sum_{k \neq i} I_k$$

여기서 $C$는 억제 포화 전압이 수동 포화 전압보다 낮은 정도를 나타냅니다.

### 핵심 수식: 적응 수준을 포함한 평형 활동

평형($\frac{d}{dt}x_i = 0$)에서 도출 과정:

$$0 = -Ax_i + (B - x_i)I_i - (x_i + C)\sum_{k \neq i} I_k$$

$J = \sum_{k \neq i} I_k$, $I = I_i + J$로 놓고 전개합니다:

$$0 = -Ax_i + BI_i - x_i I_i - x_i J - CJ = BI_i - CJ - x_i(A + I)$$

$$x_i = \frac{BI_i - CJ}{A + I}$$

여기서 $J = I - I_i$이므로 $BI_i - CJ = BI_i - C(I - I_i) = (B+C)I_i - CI$를 대입하면:

$$x_i = \frac{(B+C)I_i - CI}{A+I} = \frac{(B+C)I}{A+I}\left[\frac{I_i}{I} - \frac{C}{B+C}\right]$$

최종 결과:

$$x_i = \frac{(B+C)I}{A+I}\left[\theta_i - \frac{C}{B+C}\right]$$

#### 각 부분의 의미

| 부분 | 의미 | 역할 |
|------|------|------|
| $\frac{(B+C)I}{A+I}$ | **Weber 법칙 항** | 총 입력 $I = \sum I_k$에 대한 정규화 |
| $\theta_i = \frac{I_i}{I}$ | **반사율 (reflectance)** | 전체 입력 중 $i$번째 셀의 비율 |
| $\frac{C}{B+C}$ | **적응 수준 (adaptation level)** | 활동이 양수가 되려면 넘어야 할 문턱값 |

> **해석**: 활동 $x_i$는 "Weber 법칙에 의한 크기 조절" × "[반사율 − 적응 수준]"로 분해됩니다.
> - $\theta_i > \frac{C}{B+C}$이면: 활동 양수 (흥분)
> - $\theta_i < \frac{C}{B+C}$이면: 활동 음수 (과분극, hyperpolarization)
> - $\theta_i = \frac{C}{B+C}$이면: 활동 = 0

이 구조는 심리물리학의 **적응 수준 이론(Adaptation Level Theory, Helson 1964)** 과 수학적으로 대응됩니다. Helson은 실험 데이터에서 귀납적으로, 인간이 자극을 절대값이 아니라 **최근 경험의 가중 평균**을 기준으로 지각한다는 것을 발견했습니다 — 예컨대 같은 무게도 이전에 들었던 물체에 따라 다르게 느껴집니다. Grossberg의 네트워크는 그 "기준"이 단순히 심리적 경향이 아니라, $V^- < V^p$라는 생물물리학적 비대칭에서 수학적으로 도출됨을 보여줍니다.

### 깨진 대칭 (Broken Symmetry): B ≫ C (Figure 2.29)

> **"Adaptation levels require a broken symmetry between excitation and inhibition."**
> (적응 수준은 흥분과 억제 사이의 깨진 대칭을 요구한다)

적응 수준이 존재한다는 사실은 즉각적으로 중요한 질문을 제기합니다: 충분히 많은 비율($\theta_i$)이 적응 수준보다 커서 양수 활동을 가지려면 어떤 조건이 필요한가? 이 질문의 답이 바로 흥분과 억제 사이의 **깨진 대칭(broken symmetry)** — 흥분이 억제보다 유리한 비대칭 구조 — 입니다.

<img src="images/fig2_29.jpeg" alt="Figure 2.29 — 적응 수준과 깨진 대칭" width="50%">

$\theta_i$는 정의상 1보다 작고, 모든 비율의 합은 1입니다. 적응 수준 $\frac{C}{B+C}$도 1보다 작아야 하며, 충분히 많은 비율이 양수 활동을 가지려면 $B \gg C$이어야 합니다.

이는 **흥분-억제 간의 대칭이 깨진(broken symmetry)** 상태입니다:
- 흥분 범위: $[0, B]$ — **넓음**
- 억제 범위: $[-C, 0]$ — **좁음**
- $V^+ \gg V^-$이므로 흥분이 유리

### 깨진 대칭이 조절하는 것: 동시에 켜지는 셀의 수

깨진 대칭을 "모든 셀이 전부 켜지거나 꺼진다"는 뜻으로 오해하기 쉽습니다. 하지만 이것은 winner-take-all 현상으로, 별개의 이야기입니다. 깨진 대칭이 실제로 조절하는 것은 **적응 수준의 위치** — 즉 "몇 개의 셀이 동시에 양수 활동을 가질 수 있느냐"입니다.

적응 수준 $\frac{C}{B+C}$가 어디 있느냐에 따라 결과가 달라집니다:

| $B$와 $C$의 관계 | 적응 수준 | 결과 |
|-----------------|----------|------|
| $B = C$ (대칭) | $1/2$ | $\theta_i > 1/2$인 셀만 양수. $\sum \theta_i = 1$이므로 보통 **1개 셀**만 활성 → winner-take-all |
| $B \gg C$ (깨진 대칭) | $\approx 0$ | 평균 이상인 셀 다수가 양수 → **분산 표현(distributed representation)** |
| $B = (n-1)C$ (노이즈 억제 최적) | $1/n$ | 균일 패턴만 소거, 나머지 패턴은 살아남음 |

정리하면:
- 대칭($B = C$): 네트워크가 하나의 승자만 선택 → **경쟁(competition)** 극대화
- 비대칭($B \gg C$): 많은 셀이 동시에 활성 → **협력(cooperation)** 허용
- 최적값($B = (n-1)C$): 그 중간 — 균일한 노이즈는 억제하되, 의미있는 패턴은 유지

> 이 깨진 대칭은 뇌 전체에 편재하며, 다음 절에서 설명할 **노이즈 억제** 등 중요한 기능적 결과를 낳습니다.

## 6. 정보적 노이즈 억제 (Figure 2.30)

> **"Informational noise suppression and the selection of potentially informative activity patterns."**
> (정보적 노이즈 억제와 잠재적으로 유의미한 활동 패턴의 선별)

깨진 대칭에서 즉각적으로 도출되는 매우 중요한 성질이 있습니다. 그것은 **노이즈가 유용한 정보 처리를 오염시키기 전에 억제된다**는 것입니다. Grossberg는 이미 순환 shunting off-surround 네트워크에서의 노이즈 억제(작은 준랜덤 내생 활성화 억제)를 언급한 바 있지만, 여기서는 또 다른 잠재적으로 재앙적인 종류의 노이즈 — **정보적 노이즈(informational noise)** — 를 소개합니다.

### 문제: 정보 없는 균일 입력

만약 네트워크의 모든 셀에 **동일한 총 입력**이 들어온다면 (균일 패턴, 공간 주파수 = 0), 이 입력은 **정보가 없습니다**. 어떤 특징도 다른 특징보다 우세하지 않기 때문입니다.

이런 무정보 입력이 학습을 촉발한다면 재앙적입니다 — 기존에 학습된 모든 특징 선택적 LTM(장기 기억) 흔적이 지워질 수 있습니다.

### 해결: 적응 수준을 통한 자동 소거

<img src="images/fig2_30.jpeg" alt="Figure 2.30 — 정보적 노이즈 억제: 적응 수준 선택" width="50%">

깨진 대칭이 이 문제를 우아하게 해결합니다. $n$개의 셀이 있는 네트워크에서:

**적응 수준 설정**: $B = (n-1)C$, 즉 $\frac{C}{B+C} = \frac{1}{n}$

**균일 입력 패턴 선택**: 모든 $\theta_i = \frac{1}{n}$ (각 셀이 전체의 $1/n$씩 동일하게 받음)

수식에 대입하면:

$$x_i = \frac{(B+C)I}{A+I}\left[\frac{1}{n} - \frac{1}{n}\right] = 0$$

> **결론**: 입력이 아무리 강해도, 균일 패턴이면 모든 셀의 활동이 **정확히 0**이 됩니다. 입력 강도($I$)에 전혀 무관합니다!

반면, 균일하지 않은 패턴($\theta_i \neq 1/n$인 셀이 존재)에서는:
- $\theta_i > 1/n$인 셀: 양수 활동 (해당 특징이 평균보다 강함)
- $\theta_i < 1/n$인 셀: 과분극 (해당 특징이 평균보다 약함)

이것이 **정보적 노이즈 억제(informational noise suppression)** 입니다.

> **AGC만으로는 부족하다**: "균일 패턴 소거"는 자동 이득 제어(shunting)만으로는 나타나지 않습니다. Section 1-3의 침묵 억제 네트워크($C = 0$)에서 균일 입력을 넣으면:
>
> $$x_i = \frac{BI_i}{A+I} = \frac{B}{n} \cdot \frac{I}{A+I} \neq 0$$
>
> 출력이 있습니다. AGC는 비율을 보존하고 스케일을 정규화할 뿐, 균일 패턴을 소거하지 않습니다. 소거가 가능해지려면 반드시 **과분극($C > 0$)** — 즉 $V^- < V^p$ — 이 필요합니다. 과분극이 적응 수준이라는 문턱값을 만들고, 그 문턱값이 균일 패턴과 정확히 일치할 때 소거가 일어납니다.

## 7. 노이즈 억제에서 패턴 매칭으로 (Figures 2.31-2.32)

> **"Informational noise suppression and bottom-up/top-down pattern matching."**
> (정보적 노이즈 억제와 상향/하향 패턴 매칭)

### 왜 갑자기 "상향/하향"이 등장하는가?

지금까지 네트워크는 감각 입력($I_i$, 상향) 하나만 받았습니다. 그런데 실제 뇌에는 감각 경로와 거의 같은 수의 **하향(top-down) 피드백 연결**이 있습니다 — 고차 영역이 저차 영역으로 신호를 되돌려 보냅니다. 이 하향 신호는 뇌가 현재 갖고 있는 **기대(expectation)** 또는 **예측**을 나타냅니다.

"기대와 실제 입력이 만났을 때 어떤 일이 일어나야 하는가?"가 이 절의 질문입니다. 별도의 매칭 메커니즘이 필요할까요? Grossberg의 발견은 그렇지 않다는 것입니다 — 이미 확립된 노이즈 억제 원리가 그 역할을 그대로 수행합니다.

앞 절에서 밝힌 것은 이것입니다: **균일한 패턴($\theta_i = 1/n$)은 자동으로 억제된다.** 이제 한 가지 질문을 던져봅시다:

> 두 패턴이 동시에 들어올 때, 어떤 경우에 그 합이 균일해지고, 어떤 경우에 균일하지 않게 될까?

- 합이 **균일** → 억제 → 두 패턴이 **불일치(mismatch)**
- 합이 **불균일** → 생존 → 두 패턴이 **일치(match)**

이것이 노이즈 억제에서 패턴 매칭으로의 핵심 도약입니다. "균일한 것은 억제한다"는 규칙이 그대로 "불일치하는 쌍은 억제하고, 일치하는 쌍은 증폭한다"는 매칭 원리로 작동합니다. 별도의 메커니즘이 필요하지 않습니다 — 같은 수학이 두 역할을 동시에 합니다.

### 위상 불일치 = 노이즈 억제

Figure 2.31 상단: 두 입력 패턴이 서로 **위상이 반대(out of phase)** 인 경우를 생각합니다.
- 패턴 A: 왼쪽이 크고 오른쪽이 작음
- 패턴 B: 왼쪽이 작고 오른쪽이 큼

이 둘을 더하면? → **균일한 패턴** → 정보적 노이즈로서 **억제됨**!

> **의미**: 서로 불일치하는(mismatched) 패턴들은 합쳐졌을 때 자동으로 소거됩니다.

<img src="images/fig2_31.jpeg" alt="Figure 2.31 — 노이즈 억제에서 패턴 매칭으로" width="50%">

### 위상 일치 = 패턴 증폭

Figure 2.31 하단: 두 입력 패턴이 서로 **위상이 같은(in phase)** 경우.
- 패턴 A: 왼쪽이 크고 오른쪽이 작음
- 패턴 B: 역시 왼쪽이 크고 오른쪽이 작음

이 둘을 더하면? → **원래 패턴이 더 강화됨** → shunting 역학에 의해 **자동 이득 제어로 증폭**!

### 핵심 수식: 상향(bottom-up) + 하향(top-down) 입력 결합 (Figure 2.32)

<img src="images/fig2_32.jpeg" alt="Figure 2.32 — 공명의 기반: 매칭된 패턴의 증폭" width="50%">

상향 입력 $I_i$와 하향 입력 $J_i$가 동시에 들어올 때:

$$x_i = \frac{(B+C)(I+J)}{A+(I+J)}\left[\theta_i - \frac{C}{B+C}\right]$$

여기서:
- $I = \sum_i I_i$: 상향 입력의 총합
- $J = \sum_i J_i$: 하향 입력의 총합
- $\theta_i = \frac{I_i + J_i}{I + J}$: **결합된 반사율** (상향 + 하향 입력의 비율)

> **ART Matching Rule의 기초**: 이 수식은 나중에 **적응 공명 이론(Adaptive Resonance Theory, ART)** 의 패턴 매칭 규칙으로 발전합니다.
>
> - 하향 기대(top-down expectation)와 상향 입력(bottom-up input)이 일치하면 → 공명(resonance) → 학습
> - 불일치하면 → 억제 → 새로운 카테고리 탐색

하향 기대는 **조절적(modulatory)** 이어야 합니다 — 단독으로 셀을 완전히 활성화하지는 못하지만, 감작(sensitize)하거나 준비(prime)시킬 수 있습니다.

## 8. 발달 과정에서의 노이즈 억제 파라미터 (Figure 2.33)

> **"How does informational noise suppression arise during development? Opposites attract!"**
> (정보적 노이즈 억제는 발달 과정에서 어떻게 출현하는가? 반대가 끌어당긴다!)

노이즈 억제 조건 $C/B = 1/(n-1)$을 얻었습니다. 그런데 여기서 당연한 질문이 생깁니다: **이 조건이 실제 뇌에서는 어떻게 충족되는가?** 어떤 설계자가 각 세포의 $B$, $C$ 값과 네트워크의 연결 수 $n$을 맞춰놓지는 않았을 텐데. 이 조건이 자연스럽게 발달 과정에서 출현할 수 있다면, 노이즈 억제는 진화적으로 "무료"로 얻어지는 성질입니다. Grossberg는 이 가능성을 수식을 다시 읽는 것만으로 포착합니다.

정보적 노이즈 억제가 뇌의 많은 과정에서 얼마나 유용한지를 고려하면, 이 성질이 발달 과정에서 어떻게 생겨나는지가 중요한 질문이 됩니다. 핵심적인 단서 — 실제로 현재 데이터를 넘어서는 예측 — 는 적응 수준 방정식을 다른 방식으로 재작성함으로써 얻어집니다.

### Opposites Attract Rule (반대가 끌어당긴다)

<img src="images/fig2_33.jpeg" alt="Figure 2.33 — Opposites Attract Rule: 세포내 ↔ 세포간 파라미터 대응" width="50%">

노이즈 억제 조건 $\frac{C}{B+C} = \frac{1}{n}$을 다시 쓰면:

$$\frac{C}{B} = \frac{1}{n-1}$$

이 등식의 **양변이 가리키는 것이 다릅니다**:

| 좌변: $C/B$ | 우변: $1/(n-1)$ |
|-------------|-----------------|
| **세포내(intracellular)** 파라미터 | **세포간(intercellular)** 파라미터 |
| $C$: 억제 포화 전압, $B$: 흥분 포화 전압 | $1$: on-center 연결 수, $n-1$: off-surround 연결 수 |
| 개별 세포의 생물물리학적 특성 | 네트워크의 연결 구조 |

### 예측: 형태발생 과정에서의 대칭 깨짐

이 등식은 놀라운 **예측**을 제공합니다:

> 발달(morphogenesis) 과정에서, 세포내 흥분-억제 포화 전압의 비대칭이 세포간 흥분-억제 연결의 성장을 **유인(attract)** 하여, 양자 간 비대칭이 **일치**할 때까지 연결이 성장한다.

다시 말해:
- 세포 내부의 $V^+ \gg V^-$ (깨진 대칭)이
- 네트워크 수준에서 on-center(1개) vs off-surround($n-1$개) 연결의 비대칭을 **끌어당기며 형성**합니다

이는 실험으로 검증 가능한 예측이며, 정보적 노이즈 억제가 진화/발달 과정에서 자연스럽게 출현하는 메커니즘을 제시합니다.

## 9. 다중 채널 네트워크와 윤곽 검출 (Figures 2.34-2.36)

> **"From a global off-surround to a distance-dependent one that preserves noise suppression."**
> (전역적 off-surround에서 노이즈 억제를 보존하는 거리 의존적 off-surround로)

지금까지의 on-center off-surround는 수학적으로 편리한 이상적 모델이었습니다 — 하나의 셀이 **나머지 모든 셀**을 동등하게 억제합니다. 그런데 실제 뇌의 시각 피질이나 망막을 보면, 연결은 거리에 따라 감소합니다. 바로 옆 셀은 강하게 억제하고, 멀리 있는 셀은 약하게 억제합니다.

이 현실적 구조로 바꿨을 때 **노이즈 억제가 유지되는가?** 그리고 유지된다면 어떤 새로운 기능이 생겨나는가? 이 두 질문이 9절의 내용입니다. 결론을 미리 말하면: 거리 의존적(가우시안) 연결도 노이즈 억제를 보존하며, 그 결과로 **윤곽 검출(contour detection)** 이라는 새로운 기능이 자동으로 나타납니다.

지금까지 다룬 네트워크(Figure 2.23)는 on-center가 하나의 셀, off-surround가 나머지 모든 셀을 동등하게 억제하는 극단적 경우였습니다. 이는 실제 뇌에서 발견되는 연결 구조의 **극한 사례(limiting case)** 입니다. 이제 더 현실적인 구조로 발전시킵니다.

### 단일 채널에서 다중 채널로

<img src="images/fig2_34.jpeg" alt="Figure 2.34 — 대칭 깨짐의 역학과 해부학: 다중 채널 네트워크" width="50%">

지금까지는 한 셀이 나머지 **모든** 셀을 억제하는 "전역적" 네트워크를 다뤘습니다. 실제 뇌에서는 연결이 **거리에 따라 감소**합니다:

- **On-center**: 좁은 가우시안(Gaussian) — 가까운 셀을 흥분시킴
- **Off-surround**: 넓은 가우시안 — 더 넓은 범위의 셀을 억제

이것이 **다중 채널(multiple channel)** 네트워크입니다 (Figure 2.34).

### 핵심 수식: DOG와 SOG (Figure 2.35)

가우시안 커널을 정의합니다:

$$C_{ki} = Ce^{-\mu(k-i)^2}, \quad E_{ki} = Ee^{-\nu(k-i)^2}$$

- $C_{ki}$: on-center 가우시안 커널 (좁음, $\mu > \nu$이면)
- $E_{ki}$: off-surround 가우시안 커널 (넓음)

<img src="images/fig2_35.jpeg" alt="Figure 2.35 — 비율 대비 검출기: DOG와 SOG" width="50%">

평형 활동:

$$x_i = \frac{I\sum_{k=1}^{n}\theta_k F_{ki}}{A + I\sum_{k=1}^{n}\theta_k G_{ki}}$$

여기서:
- $F_{ki} = BC_{ki} - DE_{ki}$ — **가중 가우시안 차(weighted Difference Of Gaussians, DOG)**
- $G_{ki} = C_{ki} + E_{ki}$ — **가우시안 합(Sum Of Gaussians, SOG)**

#### DOG / SOG의 역할

| 연산 | 위치 | 기능 |
|------|------|------|
| DOG ($F_{ki}$) | 분자 | 입력의 **대비(contrast)** 를 계산 — 가장자리(edge) 검출 |
| SOG ($G_{ki}$) | 분모 | 대비를 **정규화** — 비율 대비(ratio contrast) 실현 |

### 윤곽 검출 조건 (Figure 2.36)

> **"Informational noise suppression implies contour detection in a multiple scale network."**
> (정보적 노이즈 억제는 다중 스케일 네트워크에서 윤곽 검출을 함축한다)

DOG를 SOG로 나누면 각 위치에서의 입력 **대비(contrast)** 를 계산합니다. 네트워크가 정보적 노이즈도 억제하도록 보정하면, 입력 크기가 공간적으로 충분히 빠르게 변하는 위치에서만 셀이 선택적으로 반응합니다. 이는 시각 이미지에서 충분히 강한 밝기 경계, 텍스처, 음영이 있는 위치에 해당합니다. 이 성질은 3장 이후의 모든 시각 설명을 이해하는 데 필수적입니다.

<img src="images/fig2_36.jpeg" alt="Figure 2.36 — 노이즈 억제와 윤곽 검출" width="50%">

정보적 노이즈 억제가 다중 채널에서도 작동하려면:

$$B\sum_{k=1}^{n} C_{ki} \leq D\sum_{k=1}^{n} E_{ki}$$

이 부등식이 성립하면:
- **균일 패턴** ($\theta_k = 1/n$ for all $k$) → $x_i \leq 0$ → **억제됨**
- **입력이 급변하는 위치** (가장자리, 텍스처, 음영 경계) → $x_i > 0$ → **선택적 활성화**

> **결과**: 이 네트워크는 **비율 대비 검출기(ratio contrast detector)** 로 기능합니다:
> - **반사율(reflectance)**: 물체 표면의 고유 밝기
> - **색상(color)**: 파장별 반사 비율
> - **표면 윤곽(surface contour)**: 깊이나 방향 변화
>
> 조명(illuminant)이 바뀌어도 비율은 보존되므로, 조명 할인(discount the illuminant) 효과가 자동으로 실현됩니다.

Figure 2.36의 예시:
- 직사각형 입력 $I_i$ → 입력이 0인 곳과 상수인 곳 모두 $x_i = 0$
- 입력이 **급변하는 경계**에서만 $x_i > 0$ → 윤곽만 검출

## 10. 뇌를 진화시키는 방법: 모델링 방법론 (Figure 2.37)

9절까지의 내용은 모두 **하나의 출발점**에서 시작했습니다: "유한한 뉴런으로 무한한 입력 범위를 어떻게 처리하는가?" 이 질문 하나에서 막 방정식, Weber 법칙, 과분극, 노이즈 억제, 패턴 매칭, 윤곽 검출이 연쇄적으로 도출되었습니다. Grossberg는 이제 메타 수준의 질문을 던집니다: **어떻게 이런 연쇄가 가능했는가?** 그리고 이 방법을 체계화하면 다른 뇌 기능에도 같은 방식으로 접근할 수 있지 않을까?

### 모델링 사이클

<img src="images/fig2_37.jpeg" alt="Figure 2.37 — 모델링 방법론과 사이클" width="50%">

Grossberg의 이론적 방법론은 1957년부터 사용해 온 **순환적 모델링 과정**입니다:

```
행동 데이터 (Behavioral Data)
    ↓ Art of Modeling (정적 데이터에서 실시간 역학 추론)
설계 원리 (Design Principles)
    ↓ Method of Minimal Anatomies (Occam's Razor)
수학 모델 (Mathematical Model and Analysis)
    ↓ 시뮬레이션 및 분석
예측 (Brain Predictions + Behavioral Predictions)
    ↓ 실험 데이터로 검증
행동 데이터 (더 큰 데이터베이스) → ... 반복
```

### 핵심 방법론 원칙

#### 1. Art of Modeling (모델링의 기술)

"정적 데이터에서 실시간 역학을 추론한다"는 것이 구체적으로 무슨 뜻인가? 예를 들어, 실험자가 여러 배경 조명 수준에서 셀의 반응 곡선을 측정했다고 합시다 — 이것은 **정적(static) 입출력 데이터**입니다. 그런데 Grossberg는 이 곡선이 평형 상태의 스냅샷이라는 것을 알고 있습니다. 평형이란 $\frac{d}{dt}x_i = 0$이므로, 관찰된 곡선의 형태로부터 어떤 미분 방정식이 그 평형을 만들어내는지를 **역산(infer)** 할 수 있습니다. 즉, 정적 측정 → 동적 방정식의 형태를 결정 → 방정식의 실시간 거동을 예측. 이것이 "시간 속에서 생각하는 기술"입니다.

#### 2. Method of Minimal Anatomies (최소 해부학 방법)

이 장에서 우리가 실제로 사용한 방법입니다. 각 구성 요소를 제거하면 무슨 일이 생기는지를 보면 "왜 필요한가"가 명확해집니다:

| 제거 대상 | 결과 |
|-----------|------|
| Off-surround | 포화 문제 재발, 비율 계산 불가 |
| 과분극 ($C=0$) | 적응 수준 없음, 노이즈 억제 불가 |
| 깨진 대칭 ($B=C$) | 대부분의 셀이 과분극, winner-take-all로 붕괴 |
| 가우시안 거리 의존성 | 윤곽 검출 불가, 공간 정보 손실 |

각 요소가 제거되면 설명할 수 없는 현상이 생기므로 모두 "최소"에 해당합니다. 더 추가할 이유가 없는 한 추가하지 않습니다.

#### 3. 왜 행동 데이터에서 출발하면 항상 뇌처럼 보이는가?

> 행동 데이터에서 출발하여 도출된 모델은 **항상 뇌처럼 보인다**.

이것이 우연이 아닌 이유: 뇌는 수억 년간 실세계 문제(noise-saturation, 패턴 인식, 적응)를 해결하기 위해 진화했습니다. Grossberg는 같은 문제를 수학적으로 해결하려 합니다. **같은 문제에 대한 최소 해는 같을 수밖에 없습니다.** 행동 데이터 → 설계 원리 → 최소 모델의 경로가 진화 → 자연선택 → 신경 구조의 경로와 평행하게 수렴합니다.

## 11. 임베딩, 언럼핑, 대응 원리

모델링 사이클을 계속 돌리면 모델이 점점 정교해집니다. 그런데 이 정교화 과정에는 규칙이 있어야 합니다 — 새 모델이 기존 모델과 단절되면 과거의 설명력이 날아가고, 누적이 되지 않습니다. 물리학에서 뉴턴 역학 → 상대성이론 → 양자역학으로 넘어갈 때 각 이론이 이전 이론을 **극한 사례로 포함**했듯이, 뇌 이론도 같은 원칙이 필요합니다. Grossberg는 이를 세 원리로 정리합니다.

### 세 가지 모델 진화 원리

모델이 점진적으로 정교해지는 과정을 세 가지 상호 보완적 원리로 설명합니다:

#### 1. 임베딩 원리 (Embedding Principle)
> 새로운 모델은 이전 모델을 **포함(embed)** 해야 한다.

이 장에서 실제로 일어난 것을 보면 바로 이해됩니다:

- 과분극 모델($C > 0$)에서 $C = 0$으로 놓으면 → 침묵 억제 모델이 그대로 나옴
- 가우시안 on-center off-surround에서 $\mu \to \infty$, $\nu \to \infty$ 극한 → 단일 셀 모델로 수렴
- 7절의 상향+하향 결합 수식에서 $J_i = 0$으로 놓으면 → 5절의 단독 상향 입력 수식이 나옴

새 파라미터를 특정 값으로 설정하면 이전 모델이 **특수 사례**로 나와야 합니다. 만약 그렇지 않다면, 이전 모델이 틀린 것이거나 새 모델이 잘못 설계된 것입니다. Grossberg의 1957년 이래 모든 모델이 이 원리를 만족했습니다(한 번도 폐기된 적 없음).

Grossberg는 자신의 초기 모델을 **Embedding Fields**라 명명했습니다 (Grossberg, 1964). "Embedding"은 임베딩 제약을, "Fields"는 신경망 내 상호작용이 행동에 매핑되는 창발적 특성을 의미합니다.

#### 2. 언럼핑 (Unlumping)
> 뭉쳐진(lumped) 변수를 더 세분화된(finer) 변수로 풀어놓는 과정

이 장에서의 구체적 예시:

- **침묵 억제 → 과분극**: 단일 파라미터 $g^-$가 실제로는 $V^-$와 $V^p$의 관계를 숨기고 있었음. 이를 분리하자 적응 수준이 등장
- **단일 입력 → 상향+하향**: 초기 모델의 $I_i$가 실제로는 $I_i$(감각) + $J_i$(피드백)의 합이었음. 분리하자 패턴 매칭이 등장
- **전역 off-surround → 가우시안**: 단일 억제 강도 $n$이 실제로는 거리에 따라 달라지는 함수였음. 분리하자 공간적 윤곽 검출이 등장

언럼핑할 때마다 새로운 기능이 나타납니다. 이것이 Grossberg가 "단계적 정교화"라 부르는 과정의 실체입니다.

#### 3. 대응 원리 (Correspondence Principle)
> 뭉쳐진 모델의 변수가 세분화된 모델의 변수로 어떻게 변환되는지를 명시

언럼핑 후에는 반드시 "옛 변수 = 새 변수들의 어떤 조합"인지를 보여야 합니다. 예를 들어:

$$I_i^{\text{(lumped)}} = I_i + J_i \quad \text{(상향 + 하향의 합)}$$

이것이 성립하지 않으면 두 모델은 실제로 **다른 것**을 설명하고 있는 겁니다. 많은 신경과학 이론들이 이 검증을 통과하지 못합니다 — 새 이론을 제안할 때 기존 이론이 어디에 해당하는지 보여주지 않는 것입니다.

물리학의 유비:
- **Bohr**: 양자수 $n \to \infty$ 극한에서 고전 전자기학으로 수렴
- **Einstein**: $v \ll c$ 극한에서 뉴턴 역학으로 수렴

### 모델 진화의 일관성

> 이 세 원리가 함께 작동하여, 모델의 **개념적 진화(conceptual evolution)** 가 뇌의 **생물학적 진화**를 반영하도록 보장합니다. 최소 해부학 방법은 진화 과정에서 점점 더 복잡한 환경 압력이 뇌에 반영되는 방식을 이해하는 도구입니다.

## 12. 뇌 이론에서 기술 응용으로 (Figure 2.38)

지금까지의 모든 논의는 "뇌를 이해하기 위한" 이론이었습니다. 그런데 Grossberg는 이 이론이 단순히 뇌과학의 학문적 성과로 끝나지 않는다고 주장합니다. 모델링 사이클의 **매 단계** — 행동 데이터를 설계 원리로 변환할 때, 설계 원리를 수학 모델로 만들 때, 모델에서 예측을 도출할 때 — 마다 실제 공학 문제에 적용할 수 있는 기술이 파생됩니다. 뇌 이해와 기술 개발이 같은 사이클 안에 있다는 주장입니다.

### 모델링 사이클의 모든 단계에서 기술이 파생된다

Figure 2.38은 Figure 2.37의 모델링 사이클에 **기술 응용(Technological Applications)** 이 추가된 확장판입니다:

### 기술 응용의 세 요소

| 요소 | 유래 | 의미 |
|------|------|------|
| **기능 (Function)** | 행동 분석 → 설계 원리 | 응용이 해결해야 할 **문제** |
| **메커니즘 (Mechanism)** | 설계 원리 → 신경망 모델 | 문제를 해결하는 **방법** |
| **자율성 (Autonomy)** | 신경망의 실시간 적응 설계 | 변화하는 환경에 **스스로 적응** |

### Deep Learning vs. ART: 파국적 망각 문제

이 장의 전체 내용이 사실 이 대비를 이해하는 배경입니다. 핵심 문제는 **안정성-가소성 딜레마(stability-plasticity dilemma)** 입니다:

> 새로운 것을 학습하려면 가소성(plasticity)이 필요하지만, 가소성이 너무 높으면 새 학습이 기존 학습을 덮어씁니다. 반대로 안정성(stability)이 너무 높으면 새로운 환경에 적응하지 못합니다.

**딥러닝이 파국적 망각을 겪는 이유**: 딥러닝은 가중치를 전체 예제에 걸쳐 공유합니다. 새로운 카테고리의 데이터로 역전파를 실행하면, 경사하강법이 **기존 가중치를 덮어써서** 이전에 학습한 카테고리의 성능이 붕괴합니다. 이를 피하려면 전체 데이터를 처음부터 다시 학습해야 합니다.

**ART가 이를 방지하는 이유**: 바로 이 장에서 다룬 패턴 매칭 메커니즘 덕분입니다. 새 입력이 들어왔을 때:
- 기존 카테고리와 **충분히 일치(match)** → 기존 가중치를 소폭 업데이트 (안정성 유지)
- 기존 카테고리와 **불일치(mismatch)** → 기존 가중치 건드리지 않고 **새 카테고리 생성** (가소성 발휘)

이 결정은 7절의 패턴 매칭 원리, 즉 "위상 불일치 → 억제, 위상 일치 → 증폭"과 동일한 수학에 기반합니다. 노이즈 억제 → 패턴 매칭 → 안정적 학습이 하나의 연속된 구조입니다.

| | Deep Learning | ART |
|---|---|---|
| 기반 | 역전파(backpropagation), 최급강하법 | 적응 공명 이론 |
| 수학적 뿌리 | Gauss 이래의 고전적 최적화 ("neo-classical") | 뇌에서 영감받은 비선형 역학 |
| 뇌와의 일관성 | 뇌의 메커니즘과 **불일치** | 뇌 메커니즘을 **반영** |
| 파국적 망각 | **발생** — 새 학습이 기존 학습을 덮어씀 | **방지** — 불일치 시 새 카테고리 생성 |
| 새 카테고리 | 전체 재학습 필요 | 실시간 온라인 학습 가능 |
| 적합 환경 | 통계가 안정적인 대규모 데이터셋 | 빠르게 변화하는 실세계 환경 |

> **Grossberg의 전망**: 기술 기업들이 빠르게 변화하는 빅데이터 환경에서 적응적 자율 지능을 실현해야 할 때, Deep Learning에서 ART 같은 모델로 전환하게 될 것이다.

## 요약: 전체 서사 흐름

```
사고 실험 (noise-saturation dilemma)
    ↓ 곱셈적 제어의 필요성
자동 이득 제어 + on-center off-surround
    ↓ 수학적 형식화
막 방정식 (membrane equations)
    ↓ 대수 좌표 변환
Weber 법칙 + 이동 특성
    ↓ 과분극 도입
적응 수준 + 깨진 대칭
    ↓ 균일 패턴 소거
정보적 노이즈 억제
    ↓ 상향+하향 패턴 결합
패턴 매칭 → ART의 기초
    ↓ 가우시안 커널 도입
다중 채널 윤곽 검출 (DOG/SOG)
    ↓ 메타 수준
모델링 방법론 (임베딩, 언럼핑, 대응)
    ↓ 실세계 적용
뇌 이론 → 기술 응용
```

이 서사는 하나의 단순한 문제(noise-saturation dilemma)에서 출발하여, 각 단계마다 **하나의 작은 변형**을 추가함으로써, 뇌의 근본적인 계산 원리를 점진적으로 밝혀가는 과정입니다. Grossberg는 이 방법론 자체가 뇌의 진화 과정을 반영한다고 주장합니다.
